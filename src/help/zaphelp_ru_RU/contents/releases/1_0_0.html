Версия 1.0.0

В этой версии были сделаны следующие изменения:

Существенные изменения:

Справка

Справка была добавлена для всех функциональных возможностей, предоставляемых в настоящее время в ZAP.
Контекстно-зависимая справка также была выполнена так, что при нажатии клавиши F1, будет отображаться экран справки, связанный с вкладкой, которую вы выбрали.

Точки остановки

Использование Точек остановки было изменено, чтобы быть ближе к предоставляемым в IDE.
Управление точками остановками было премещено в новую Панель инструментов верхнего уровня.
Точки останова также могут быть установлены с помощью щелчков правой кнопкой мыши в Сайты и Истории вкладки.

Версия отчета

В Версия отчета значительный изменилось и теперь включает в себя следующие пункты меню:
    Создать отчет в формате XML...Которая теперь генерирует отчеты по требованию
    Экспортировать сообщения в файл...Переехал в меню файл
    Экспортировать ответ в файл...Переехал в меню файл
    Экспортировать все URL-адреса в файл...Режим редактора
    Сравнить с другой сессией...Режим редактора

Улучшенная истории поиска

Новый Искать позволяет вам искать для регулярных выражений во всех URL-адресов, запросов и ответов.
Следующей и предыдущей кнопки позволяют вам просматривать найденные Поиск терминов в Запрос и Ответ вкладки.

Истории фильтр

Панель инструментов фильтрации истории была добавлена которая позволяет ограничить, какие запросы отображаются.
Запросы могут быть отфильтрованы по HTTP-методы, результат коды HTTP и любой ярлыки, оповещения, или заметки это были применены.

Заметки

Теперь можно связать заметки с любой просьбой.

В несколько страниц

Теперь можно связать несколько ярлыки с любой просьбой.

Больше контроля над оповещения

В Диалоговое окно «Добавить предупреждение» позволяет добавить или изменить оповещение вручную.

Новый кодер/декодер

Новый Кодировать/декодировать/хэшировать заменил предыдущий Кодер / декодер.

Пассивный сканер

Пассивный сканер превышен.
В этом выпуске Зап будет использовать только пассивная проверка для автоматического добавления ярлыки.
В будущих выпусках он также может оповещения для потенциальных проблем.

Незначительные изменения:

Панель инструментов верхнего уровня

Панель инструментов верхнего уровня была добавлена которая содержит набор кнопок для часто используемых функциональность.

Оповещения нижний колонтитул резюме

Количество высокий, средний, низкий и информационные оповещения теперь отображаются в нижний колонтитул.

Новый внешний вид и

Используется новый вид «Нимбус».

Пароль прокси обработка

На экране настройки соединения имеет новый параметр «Запрашивать учетные данные прокси-сервера при запуске».
Если выбран этот параметр, то ваш пароль прокси-сервера будет не храниться на жестком диске.

Отправить на вкладке сайты

В Закладка сайты появился новый щелкните правой кнопкой мыши элемент меню, который позволяет вам отправить запрос после внесения любых изменений в то, что вы хотите.

Каталог пользователей между сессиями

Последний каталог, используемый пользователем теперь хранится в файле конфигурации, так что оно сохраняется между сессиями.
