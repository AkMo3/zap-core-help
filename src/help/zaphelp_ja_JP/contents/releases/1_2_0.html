リリース 1.2.0

本リリースでは以下のような変更が行われました。

重要な変更

メモリ リーク

Memory leaks have been fixed in the 動的スキャナ 及び スパイダー.

アプリケーションの呼び出し

外部アプリケーションがサイトおよび履歴タブから呼び出せるようになりました。

静的スキャナ

　 静的スキャナ now looks for vulnerabilities, such as:
    Autocomplete forms with password fields
    Cookies without the 'HttpOnly' flag
    SSL Cookies without the 'secure' flag
    Weak authentication

小変更：

XML レポート

A new 'Generate XML Report...' menu item is now included in the top level Reports menu.

Manual Request Editor and Resend dialogs

Requests submitted by the 手動リクエスト 及び 再送信 dialogs are now shown in the サイト 及び 履歴 tabs.
A new 'Method' pull down allows you to switch between the HTTP methods, this automatically moves parameters between the URL and the body when a POST method is selected or deselected.

アラート UI

　 サイト tab now shows any alerts as flags to the right of any node names.
The alert counts in the フッター now show the number of different types of alerts rather than the total number of instances.

動的スキャンの遅延設定

The delay in milliseconds between each 動的スキャナ request can now be set via the オプション 動的スキャン画面. This will increase the time an active scan takes but will reduce the load on the target.

UIの変更

　 サイト tab now takes up all of the left hand side - this can be changed back via the Options Display screen if required.

The 'toolbar' on the リクエスト, レスポンス 及び ブレーク tabs and the 手動リクエスト 及び 再送信 dialogs is now at the top rather than the bottom.

The scanner counts in the フッター are now displayed on the right hand side.
