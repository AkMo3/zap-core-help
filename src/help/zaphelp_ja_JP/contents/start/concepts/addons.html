アドオン

アドオンによって、ZAPに機能を追加することができます。
アドオンはZAPの内部機能を完全に利用することによって、非常に強力な新機能を提供することが可能です。
アドオンの管理ダイアログから、オンラインのアドオンマーケットプレースにあるアドオンを動的にインストールすることができます。
通常、ZAP UIへのアドオンの追加と削除は再起動なしに行うことができます。

アドオンは、ヘルプファイルと動的に統合される文脈依存なヘルプを提供することができます。
あなたがもしこのページをZAPヘルプファイルを通して読んでいるなら(オンラインで読んでいるのではなく)、インストールしたアドオンに関連付けられたどんなヘルプページでもここで読むことができます。.

アドオンは以下のステータスのどれかが割り当てられます。
     Releaseアドオンが高い品質であることと、目的を満たしていることを示します。
     Betaアドオンがある程度の品質ではあるものの、完全ではないもしくは更にテストが必要であることを示します。
     Alphaアドオンが開発の初期段階であることを示します。

Location of add-ons available to ZAP

To make an add-on available to ZAP it must be in one of the following locations:
  • plugin directory located in the ZAP's installation folder;
  • plugin directory located in the ZAP's user folder.
