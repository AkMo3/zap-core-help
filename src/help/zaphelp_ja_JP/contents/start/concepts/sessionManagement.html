セッション管理

ZAPはWEBサイトやWEBアプリで使用できる複数の種類のセッション管理 (セッション マネジメント メソッド と呼ばれる) を取り扱っています。 Each Context has a Session Management Method defined which dictates how sessions are kept.

これまでのところ、単にcookieベースのセッション管理方法が実装されていますが、ユーザーのニーズに応じて簡単に新しいメソッドの追加をシステムはサポートします。

クッキーベースセッション管理

このメソッドの場合、セッションクッキーを介して追跡されます。現在、セッショントークンはHttpセッション 拡張からインポートされて使用されます。
