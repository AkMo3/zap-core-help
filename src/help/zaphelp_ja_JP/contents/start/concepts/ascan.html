動的スキャン

動的スキャンは選択されたターゲットに対して既知の攻撃を使い、潜在的な脆弱性を検出しようとします。

動的スキャンはこれらのターゲットへの攻撃です。
あなた自身のものでないWEBアプリケーションに対して使用しないでください。

動的スキャンは特定の脆弱性のみを見つける事が出来る事に注意する必要があります。
アクセス制御が壊れているような、論理的脆弱性はどんな動的、自動脆弱性スキャンでも見つける事が出来ないでしょう。
全ての種類の脆弱性を見つけるためには、動的スキャンに加えて、手動のペネトレーションテストを常に実行する必要があります。

Active scanning is configured using the Options Active Scan screen.
The rules that run are configured via Scan Policies - you can have as many of these as you like.
