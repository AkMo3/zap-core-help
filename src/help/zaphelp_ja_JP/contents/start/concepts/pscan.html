静的スキャン

ZAPはテストするWEBアプリケーションからの全てのレスポンスを受動的（静的）にスキャンします。
静的スキャンはとにかくレスポンスを変更する事は無く、それゆえ使用するのに安全です。
スキャンはアプリケーションの探索を遅くする事が無いようにするため、バックグラウンドスレッドで実行されます。

このリリースで、ZAPの静的スキャンは自動的にタグを追加、潜在的な問題のアラートをあげるために使用されます。

A set of rules for automatic tagging are provided by default.
These can be changed, deleted or added to via the Options Passive Scan Tags screen.

静的スキャンによって発生するアラートは静的スキャンルールオプション画面 を使用して設定します。.
