グローバルに除外されたURL

注： この機能はベータ版です。

グローバルに除外されたURLはZAPがアプリケーション全体で完全に無視する正規表現 (regex) となります。 これらのURLはZAPのプロキシ、スキャナー、またはスパイダーで表示される事はありません。 In addition, the URL regexs will be saved in the global configuration for ZAP and not inside of the session file. These URLs will persist between each use of ZAP.

The regex for URLs are configured using the Global Exclude URL screen in the ZAP Options.
