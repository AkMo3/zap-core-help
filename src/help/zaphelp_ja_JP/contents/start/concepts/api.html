API

ZAPはApplication Programming Interface (API) を提供しており、ZAPとプログラムで対話する事を可能としています。

APIはJSON、HTML、XML形式が利用可能です。
APIドキュメントはZAPをプロキシを介し、URL http://zap/ 通して利用する事が可能です。

既定では無効になっているため、APIに組み込まれているのは最小限のセキュリティである事に注意してください。
有効にした場合、APIはZAPをプロキシとして使用できる全てのマシンで利用可能となります。
既定では ZAPは「localhost」のみリスンしており、ホストマシンからのみ使用する事が出来ます。

APIは動的スキャナーやスパイダーなどのZAPのコア機能へのアクセスを提供しています。
ZAPの将来のバージョンでは、APIを通して利用可能な機能を増やすつもりです。

APIはAPI オプション画面 を使用して設定します。.
