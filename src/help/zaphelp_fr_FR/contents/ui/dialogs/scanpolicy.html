Scan Policy dialog

This allows you to enable and disable the rules that are run when passive scanning and performing an active scan.
The first screen allows you to define the default levels as well as the levels for all of the rules in a specific category.
It also lets you to load and save scan policies, allowing you to maintain a set of policies that work well in different circumstances.

The category screens allow you to define the levels for every individual rule.

Thresholds and Strengths are explained in the Options Active Scan section.
