Manajemen Sesi

ZAP menangani beberapa jenis sesi pengelolaan (disebut Metode Sesi Pengelolaan) itu bisa digunakan untuk situs web / aplikasi web. Setiap Konteks memiliki penjelasan Metode Sesi Pengelolaan yang menentukan bagaimana sesi disimpan.

Sejauh ini, hanya metode sesi pengelolaan terbukti berdasarkan cookie dan HTTP diimplementasikan, namun sistem ini mendukung penambahan metode baru yang mudah, sesuai kebutuhan pengguna.

Sesi Pengeolaan Berdasarkan Cookie

Dalam kasus metode ini sesi dilacak cookie. Saat ini, token sesi yang digunakan diimpor dari sebuah HTTP Sesi Perluasan.

HTTP Pembuktian Sesi Pengelolaan

Dalam kasus metode ini sesi dikelola dengan permintaan header HTTP Otorisasi.
