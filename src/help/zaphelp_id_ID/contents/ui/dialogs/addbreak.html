Tambahkan Break Point dialog

Ini mengizinkan anda untuk mengatur break point pada URL.

Titik istirahat didefinisikan melalui ekspresi regex.
Jika Anda mengunjungi URL yang cocok dengan ekspresi ini kemudian ZAP akan mencegat itu dan memungkinkan Anda untuk mengubah baik permintaan dan/atau tanggapan.

Diakses melalui

     Tab Sejarah'Break...' kanan klik menu item
     Tab situs'Break...' kanan klik menu item

Lihat juga

     Ikhtisar UIuntuk Ikhtisar antarmuka pengguna
     Dialoguntuk rincian dari dialog atau popup
