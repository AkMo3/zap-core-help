Options dialog

The Options dialog allows you to configure ZAP.
It include the following screens:
    Scan Aktif
    Active Scan Input Vectors
    Token Anti Anti CSRF
    API
    Otentikasi
    Breakpoints
    Sertifikat
    Check for Updates
    Koneksi
    Database
    Tampilan
    Dynamic SSL Certificate
    Fuzzer
    Keyboard
    Proxy Lokal
    Scan Pasif
    Telusuri
    Spider

Accessed via

     Top level Tools menu'Options ...' menu item
