Pilihan layar JVM

Layar ini memungkinkan Anda untuk mengkonfigurasi opsi JVM yang digunakan saat memulai ZAP.

Pilihan JVM

Bidang teks format bebas yang akan ditambahkan ke Java command line call saat memohon ZAP melalui zap.sh atau zap.bat.

Opsi itu ditambahkan sehingga ukuran kapasitas alokasi memori maksimum Java bisa diatur, yang berbentuk: -Xmxn dimana n adalah ukuran dalam byte.
Nilai-nilai yang bagus untuk bidang ini bisa jadi:

  • -Xmx256m
  • -Xmx512m
  • -Xmx1024m

Berbeda dengan opsi ZAP lainnya yang tersimpan dalam file ".ZAP_JVM.properties" di direktori ZAP default pengguna, yang tergantung pada OS yang digunakan:

  • Windows 7/8: C:\Users\<username>\OWASP ZAP
  • Windows XP: C:\Documents and Settings\<username>\OWASP ZAP
  • Linux: ~/.ZAP
  • Mac OS: ~/Library/Application Support/ZAP
Jika Anda membuat kesalahan saat mengatur opsi ini maka ZAP mungkin gagal untuk memulai.
Jika itu terjadi maka menghapus file ini harus memperbaiki masalah.

Perhatikan bahwa opsi ini tidak digunakan saat memulai ZAP via Windows exe.
