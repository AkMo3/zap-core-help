Layar Breakpoints pilihan

Layar ini memungkinkan anda untuk mengkonfigurasi break point pilihan:

Konfirmasikan jatuhnya pesan

Meminta konfirmasi ketika permintaan atau respon yang dijatuhkan.

Hanya istirahat di pesan di lingkup

Titik istirahat 'global' hanya dapat dipicu oleh permintaan yang cakupan.
Kustom istirahat poin tidak akan terpengaruh.

ZAP selalu di atas ketika breakpoint memukul

Jika dipilih ini akan berarti bahwa ZAP tetap berada di atas semua jendela lain setiap kali breakpoint dipukul.

Istirahat tombol mode

Ini mengontrol apakah ada satu gabungan tombol istirahat di Toolbar tingkat atas atau yang terpisah untuk Permintaan dan Tanggapan.
