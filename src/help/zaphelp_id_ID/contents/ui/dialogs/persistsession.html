Dialog sesi persis

Dialog ini ditampilkan secara default setiap kali Anda memulai sesi baru, yang selalu terjadi saat ZAP dimulai.
Ini memungkinkan anda untuk menentukan bagaimana anda ingin sesi ini bertahan, anda juga dapat menetapkan pilihan default sehingga anda tidak diminta lagi.

Sesi yang bertahan berarti akan disimpan dalam database lokal yang dapat anda akses di kemudian hari.
Anda tidak perlu untuk menjaga 'hemat' sesi sebagai segala sesuatu yang terjadi di sesi ini terus direkam.
Hal ini jauh lebih cepat untuk bertahan sesi di mulai, tetapi anda selalu dapat bertahan sesi nanti jika anda perlu untuk.
Jika anda menutup ZAP tanpa bertahan sesi anda maka anda tidak akan bisa mengaksesnya lagi.

Ya, saya ingin bertahan di sesi ini dengan nama berdasarkan stempel waktu saat ini

Pilih opsi ini jika anda ingin ZAP untuk menyimpan sesi di direktori default dengan nama berdasarkan waktu saat ini.

Ya, saya ingin bertahan di sesi ini tapi saya ingin menentukan nama dan lokasi

Pilih opsi ini jika anda ingin menentukan persis di mana ZAP harus menyimpan sesi.

Tidak, saya tidak ingin melanjutkan sesi ini pada saat ini

Pilih opsi ini jika anda tidak ingin ZAP untuk menyimpan sesi.
Anda dapat memilih untuk menyimpannya di lain waktu dengan menggunakan Bertahan Sesi... item menu.

Ingat pilihan saya dan jangan tanya saya lagi.

Centang kotak ini jika anda ingin ZAP untuk ingat keputusan anda dan tidak meminta anda lagi.
Anda dapat mengubah keputusan anda melalui Pilihan database layar.
