Edit menu

Menu ini menangani menemukan string di tab tertentu, mencari string di semua permintaan dan tanggapan dan mengelola sesi pelacakan.

Cari...

Ini akan membuka Menemukan dialog yang memungkinkan Anda untuk menemukan sebuah string di jendela yang dipilih saat ini.

Aktifkan Session Tracking (Cookie)

Hal ini memungkinkan rincian sesi yang disimpan dalam cookie yang akan dilacak.
Opsi ini harus dipilih untuk mengaktifkan kotak centang "Gunakan saat ini sesi pelacakan" dalam Kirim ulang dan Manual meminta Editor dialog.
Sesi pelacakan memastikan bahwa setiap permintaan yang dikirim dengan rincian sesi terbaru.
Misalnya, anda dapat merekam sesi ketika masuk sebagai salah satu pengguna dan kemudian keluar dan masuk sebagai pengguna lain.
Jika Anda mengirim permintaan dari sesi pertama tanpa sesi pelacakan maka akan menggunakan cookie dari sesi pertama.
Jika Anda mengirim permintaan yang sama dengan sesi pelacakan maka akan menggunakan cookie dari sesi kedua.

Reset Keadaan Sesi

Ini membersihkan pelacakan sesi.

Telusuri...

Ini memilih Telusur yang memungkinkan anda untuk mencari ekspresi reguler di semua URL, permintaan dan tanggapan..

Selanjutnya

Ini memilih terjadinya berikutnya string terakhir mencari.
Pesan yang relevan akan dipilih di tab Pencarian dan string yang akan ditampilkan dan disorot dalam Permintaan atau Respon tab yang sesuai.

Sebelumnya Previous

Ini memilih terjadinya sebelumnya string terakhir mencari.
Pesan yang relevan akan dipilih pada tab Pencarian dan string akan ditampilkan dan disoroti pada Permintaan atau Respon tab yang sesuai.

Perhatikan bahwa add-ons bisa menambahkan item menu tambahan.
