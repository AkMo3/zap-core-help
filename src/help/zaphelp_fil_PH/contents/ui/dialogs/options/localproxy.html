Screen ng lokal na proxy ng opsyon

Screen ng koneksyon sa mga pagpipilian ay nagpapahintulot sa iyo na magkumpigura ng mga address at port kung saan ang ZAP ay tumatanggap ng papasok na mga koneksiyon.

Lokal na Proxy

Sa pamamagitan ng default ZAP ay nakikinig sa isang lokal na tirahan at port, at ang mga ito ay mga address at port na dapat mong i-configure ang iyong browser upang gamitin bilang proxy.

Address

Ang lokal na address na gagamitin ang ZAP. Lahat ng mga magagamit na mga mensahe ay awtomatikong natukoy.

Porto

Ang port na kung saan ang ZAP ay makinig.

Likod NAT

Ay nagpapahiwatig na ang mga lokal na mga Proxy (ZAP) ay likod ng metro. Kapag ang napiling ZAP ay pagtatangka upang matukoy ang pampublikong IP address, na maayos na tiktikan at hawakan ang mga hiling sa pampublikong IP address (halimbawa, na naglingkod sa mga ZAP API).

Tandaan: Ang opsyon na ito ay suportado lamang kapag ZAP ay tumatakbo sa isang AWS EC2 pagkakataon. ZAP ay magkakamit ng public IP address mula sa AWS EC2 pagkakataon Metadatos.
ZAP dapat nagsimula sa opsyon na ito pinagana kung access sa API, sa pamamagitan ng public IP address, ay kinakailangan:

zap.sh-daemon-port 8080 - host 0.0.0.0 - kumpig proxy.behindnat=true
Gayundin, ang API pangangailangan upang maisaayos na tanggapin ng mga panlabas ng mga IP address (i.e. ang IP address mula sa kung saan ang ZAP ay pagiging na-access).

Alisin ang mga di-suportadong pagsasakodigo

Nagbibigay-daan ang proxy upang alisin ang mga di-suportadong pag-encode Batac "Tanggapin-Encoding" ulunan ng kahilingan, kaya walang (unsupported) encoding pagbabago ay ginawa upang ang tugon.
Ang opsyon na dapat maging palaging pinagana maliban kung nang pagsusuri ang encoding na pagbabago.
Ang mga mensahe na na-encode nang may di-suportadong pagsasakodigo ay hindi maging tama Nakilatis (alinman ng pasibo at aktibo scanner).

Security protocol

Ay nagbibigay-daan upang piliin ang mga bersyon ng SSL/TLS na pinagana para sa mga papasok na mga koneksiyon (halimbawa, mula sa browser). Kailangang magpagana ng hindi bababa sa isang version, hindi suportadong sa pamamagitan ng JRE mga bersyon ay napili at may kapansanan.
Dapat piliin ang mga opsyon na SSLv2Hello kaugnay ng hindi bababa sa isang bersyon ng SSL/TLS.
Ang mga opsyon sa seguridad ay tumutukoy sa parehong ang pangunahing at ang lahat ng mga karagdagang proxy.

Karagdagang proxy

Maaari kang magdagdag ng maraming karagdagang mga address at port para sa ZAP makinig sa 't gusto mo.
Ito ay maaaring maging kapaki-pakinabang kapag ang pagsusuri ng mga mobile application - maaari mong proxy ang mobile application gamit ang isang Wi-Fi interface sa parehong oras bilang gamit sa backend site na direktang gamit ang isang browser nagsisilbing proxy para sa pamamagitan ng ZAP pagtulad ng mga mobile na gumagamit ng isang agent.
