Mga opsyon ng mga Breakpoint screen

Ang screen na ito ay pinapayagan ka para i-configure ang break point na mga opsyon:

Ikomperma ang mga drop trapped na mensahe

Magtanong sa kompermasyon kung ang hiling o pagtugon ay bumaba na.

Tanging ang pagsira sa mga mensahe sa scope

Ang 'global' break point ay tanging makakapag trigger sa pamamagitan ng hiling na nasa scope.
Ang pagtangkilik sa mga break point ay hindi makakaapektado.

Ang ZAP ay laging nasa taas sa tuwing ang breakpoint ay tumatama

Kung itong napili na ibig sabihin na ang ZAP ay mananatili sa taas ng lahat ng iba pang mga window sa tuwing ang isang breakpoint ay pumalo.

Ang mode ng mga pindutan ng break

Ang mga kontrol ay may isang magkasalung break button sa Taas ng Level ng Toolbar o hiwalay ito ang isa sa kahilingan o katugunan.

Tignan din

     Pangkalahatang-ideya ng UIpara sa isang pangkalahatang-ideya ng mga gumagamit ng interface
     Mga opsyon ng dayalogopara sa mga detalye ng ibang mga opsyon na dayalogo ng mga screen
