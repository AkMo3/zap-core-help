Ang konteks ay nag papatunay sa iskren ng session

Ito ay isa sa mga Sesyon konteks iskren ito ay nag papahintulot sa iyo na pamahalaan ang mga paraan Nag pagpapatunay ito ay ginawa para sa kontekto. Tandaan Ang pagpapalit ng patunay matapos ito ay gamitin ay magiging paraan para ito ay matukoy at ito ang magiging dahilan para ito ay matanggal, at ang uri ng gumagamin ng kridensial ay kaylangan maging pareho sa pamamaraan ng nag papatunay. Matapos mapili ng paraan at uri ng pagpapatunay ang pagpipilian ay kaylangan maisaayos base sa paraan ng pagpapatunay.

Batay sa anyo ng pagpapatunay

Para maisaayos ang paraan ng pagpapatunay kaylangan mong bigyan ang login url, para mapalitan ang login Kahilingan ay tuparin, ang nilalaman ng kahilingan (POST data), Kung kinakaylangan, kilalanin ang mga parameter ginagamit para masupplayan ang 'username' at 'password'. Kung ang request body ay walang maibigay, ang login request ang gaganap dito bilang isang HTTP GET, at Kung Hindi man ang HTTP POST ang gagamitin. Base sa iskrip ng pagpapatunay

Para magamit ang paraan ng pag papatunay. Ikaw una kaylangan sumulat (and save) ng Ang iskrip ng pagpapatunay gamit ang Mga iskrip tab (tignan ang halimbawang ibinigay ng templates para sa script type at sa script tab). At kaylangan mung lagyan ng pangalan ang script sa listahan ng dropdown. Pagkatapos mung piliin ang script, kaylangan mung pindutin ang Load pindutan, naglo-load ang lahat ng hinihinge ng script. Anuman mga parameter ang tinurukoy mo bilang kinakaylangan o opsyonal sa script makikita ang pagtukoy sa interface. Ang pag papahalaga ay maaaring gamitin sa script, upang mapatunayan, ang nakakita ang mag bibigay ng halimbawa sa pagpapatunay ng mga script. Siguraduhin matapos mung gawin ang anuman pagbabago sa parameters ay hinihinge ng pagpapatunay ng script at marireload mo ang script. Kung Hindi, ang mga parameter na nakikita sa interface ay hindi na magagamit pa sa pagpapatunay at ang mga error ay maaring mangyari. Ang ang mga kredensiyal ginamit para sa bawat gumagamit Sa panahon ng otentikasiyon prosessend ganun din pinatunayan ito ng otentikasiyon script at ito ay isinaayos ng mga Mga gumagamit tab.
