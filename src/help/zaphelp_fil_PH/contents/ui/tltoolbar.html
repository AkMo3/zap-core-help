Ibabaw na Lebel ng Toolbar

Ang toolbar na ito ay nag-provide ng set ng mga kontrol para sa mga karaniwang ginagamit na functionality.

Mode pulldown

It ay nag-allow sayo sa pagbago ang kasalukuyan mode.

  Bagong sesyon

Ang button nato ay parahe sa File menu 'Bagong sesyon' menu aytem.

  Open Sesyon

Ang button na ito ang makapareha sa File menu 'Open Sesyon' menu aytem.

  Pursigidong Sesyon...

Ang button na ito ay pareha sa File menu 'Pursigidong Sesyon...' menu aytem.

  Snapshot Sesyon

Ang button na ito ay magkapareha sa File menu 'Snapshot Sesyon' menu aytem.

  Sesyon Properties...

Ang button na ito ay magkapareha sa File menu 'Properties...' menu aytem.

  Mga opsyon...

Ang button na ito ay magkapareha sa Tools menu 'Mga opsyon...' menu aytem.

  Ipakita ang lahat na tab

Ang button na ito ay magpakita ng lahat ng nakatagong mga tab.

  Itago ang hindi naka-pin na mga tab

Ang button na ito ay magtago ng lahat ng mga tab na 'unpinned'. Mga tab ay pwedeng i-pin at i-unpin via ang maliit 'pin' na icon na makita kapag ang tab ay naka-select.

  Ipakita ang mga pangalan ng tab at mga icon

Ang button na ito ay mag-toggle sa mga naka-displey na pangalan ng tab.

  I-expand ang mga site ng tab

Ito ay magbabago sa naka-displey na ang 'tree' window nakalaman ng mga Site tab mag-extend para sa buong haba ng kaliwang bahagi ng kamay.
Makabawas ito sa bilang ng espasyo na available para sa 'impormasyon' window.

  I-Expand ang Impormasyon Tabs

Ito ay magbabago ng naka-displey sa 'impormasyon' window mag-extend para sa buong haba ng ilalim.
Makabawas ito sa bilang ng espasyo na available sa 'tree' window.

  Buong Layout

Ito ay magbabago sa naka-displey para ang mga napiling tab makita sa buong screen.
Mapakinabangan ito kapag gumagamit ng ZAP sa maliit na screen.

  Mga hiling at tugon ng mga tab na magkatabi

Ito ay magbabago ng displey para ang hiling at tugon ng mga tab ay magkatabi.
Magdagdag ito ng impormasyon na pwedeng i-displey ngunit hindi mo makita ang dalawang hiling at tugon ng sabay.

  Nakita ang hiling sa ibabaw ng tugon

Ito ay magbabago ng displey para ang hiling na panel ay makita sa ibabaw ng tugon na panel.
Magbabawas ito ng impormasyon na pwedeng ma-displey ngunit maaari mong makita ang dalawang hiling at tugon ng sabay.

  Hiling at Tugon na mga panel na magkasabay

Ito ay magbabago ng displey para ang hiling na panel ay makita sa kaliwa ng tugon na panel.
Magbabawas ito ng impormasyon na pwedeng ma-displey ngunit maaari mong makita ang dalawang hiling at tugon ng sabay.

 /    I-Set / I-Unset break sa lahat ng hiling at tugon

Ito ay mag-set at mag-unset ng 'global' break point na mag-trap at mag-displey ang sunod na hiling at tugon sa loob ng Break tab.
Pwede mong baguhin ang anumang parte ng hiling o tugon na gusto mo at ipadala sa target na aplikasyon sa pamamagitan ng pagpintdut alinman 'Step' o 'Magpatuloy' buttons.
Bilang kahalili pwede mong pindutin ang 'Drop' button para matapon ang hiling o tugon.
Pwede mong magpalit sa pagitan ng isang 'combined' break button at separates ones para sa mga hiling at tugon via ang Mga opsyon breakpoints screen

 /    I-Set / I-Unset break sa lahat ng mga hiling

Mag-set at mag-unset ito ng 'global' break point na mag-trap at mag-displey sa sunod na hiling sa loob ng Break tab.
Pwede mong baguhin ang anumang parte ng hiling na gusto mo at ipadala sa target na aplikasyon sa pamamagitan ng pagpindut ng aliman 'Step' or 'Magpatuloy' buttons.
Bilang kahalili pwede mong pindutin ang 'Drop' button para matapon ang hiling.
Pwede mong palitan sa pagitan ng isang 'combined' break button at separates ones para sa mga hiling at tugon via ang Mga opsyon breakpoints screen

 /    I-Set / I-Unset break sa lahat ng mga tugon

Mag-set at mag-unset ito ng 'global' break point na mag-trap at mag-displey sa sunod na tugon sa loob ng Break tab.
Pwede mong baguhin na anumang parte ng tugon na gusto mo at at ipadala ito sa iyong browser sa pamamagitan ng pagpindut alinman 'Step' or 'Magpatuloy' buttons.
Bilang kahalili pwede mong pindutin ang 'Drop' button para matapon ang hiling.
Pwede mong palitan sa pagitan ng isang 'combined' break button at separates ones para sa mga hiling at tugon via ang Mga opsyon breakpoints screen

  Step

Pinapayagan nito ang naka-trap na hiling o tugon na magpatuloy sa aplikasyon o sa iyong browser ng kahit anong pagbabago na ginawa mo nito.
Ang 'global' break point ay manatiling set para ang sunod na hiling at tugon ay mahuli din.
Ang button na ito ay na-enable lamang kapag ang isang hiling o tugon ay na-trap.

  Magpatuloy

Ang 'global' break point ay mag-unset para sa subsequent na mga hiling at tugon ay hindi na mahuli ng ZAP maliban kung naka set ka ng break points sa partikular na URLs.
Ang button na ito ay na-enable lamang kapag ang isang hiling o tugon ay na-trap.

  Drop

Magtapon ito ng naka-trap na hiling at tugon para hindi ito mailipat sa aplikasyon o sa iyong browser.
Ang button na ito ay na-enable lamang kapag ang isang hiling o tugon ay na-trap.

  Magdagdag ng custom HTTP break point

Mag-displey ito ng Add break point dialogo na pumapayag sa iyo na mag-specify ng criteria para sa break point.

  Scan Policy Manager...

Magpakita ng Scan Policy Manager dialogo na pumapayag sa kumpigurasyon ng scan policies.

  I-Manage ang Add-ons

Magpakita ito I-Manage ang Add-ons dialogo na pumapayag sa iyo na matuklas, i-install at i-update ang add-ons mula sa online marketplace.
Pumapayag ito sa iyo na mag-uninstall ng add-ons.

 /    Napilit User Mode On / Off

Pumapalit ito ng napilitan na user mode on and off.
Ang button na ito ay ma-enable lamang kapag na-define mo ang napilitan na user kahit isa lang context, na pwedeng magawa via ang Session Contexts dialog.

I-Note na add-ons ay pwedeng magdagdag na adisyonal na mga button.
