Inilibas na 2.3.1

Ang mga pagbabagong sumusunod ay ginawa sa inilabas na ito:

  • Isyu 81 : Ang mga binago ng ZAP na kahilingan ng datos (habang nagpapalit ang mga pag-view)
  • Isyu 377 : Ang hindi natapos na mga dependensya ay naha-hang ang aktibong pagsusuri
  • Isyu 1073 : Hindi kayang tanggalin ang mga iskrip na marka bilang 'mag-load sa simula'
  • Isyu 1114 : core.newSession hidi maaaring klaruhin ang mga site
  • Isyu 1155 : Kasaysayan ng kahilingan na Tab ay hindi pinapayagan ang pag-format ng mga pagbabago
  • Isyu 1156 : Kinatawan ng gzip decoder ay hindi maaaring baguhin ang haba ng nilalaman sa pagtugon ng mga header
  • Isyu 1163 : Hindi maaaring mag-takda sa isang home directory kasama ang isang space sa linya ng nag-command
  • Isyu 1166 : Ang kalabisan ng mga index sa zapdb.iskrip
  • Isyu 1168 : Magdagdag ng kinatawan na pagsuporta para sa "deflate" na nilalaman ng pag-encode
  • Isyu 1170 : Ang ispayder ng konteksto/gumagamit mag-pop up ang mga pagpipilian ay hindi na makikita pa
  • Isyu 1179 : Hindi maaring piliin ang 2 mga kahilingan sa mga resulta sa fuzz (Ctrl + click)
  • Ang mga pahinang kahinaan ay aktibong nasuri lamang ng isang beses
  • Isyu 1182 : Mga alerto ng parehong tipo para sa magkaibang mga parametro sa parehong kahinian na pahina na nagpapakita lamang ng isang beses sa "mga alerto"
  • Isyu 1183 : NullPointerException habang pinipili ang isang node sa "mga Alerto" na tab pagkatapos burahin ang isang mensahe
  • Isyu 1191 : Cmdline na sesyon ng mga param ay walang epekto
  • Isyu 1193 : Suriin ang URL na daanan ng mga elemento - isara sa pamamagitan ng default
  • Isyu 1194 : Mga argumento ng Command line ay hindi nakapasa sa mga karugtong habang nagsisimula ang ZAP sa paraan ng daemon
  • Isyu 1196 : AbstractPlugin.bingo hindi tamang mga takda ng ebidensya sa pag atake
  • Isyu 1202 : Ang isyu na kasama ang loading na mga addon ay hindi tama ang pagsisimula
  • Isyu 1203 : Awtentikasyon ng Wordpress ng iskrip
  • Isyu 1206 : "Kasaysayan" na tab ay hindi klaro habang ang isang bagong sesyon ay ginawa sa pamamagitan ng API kasama ang ZAP sa paraan ng GUI
