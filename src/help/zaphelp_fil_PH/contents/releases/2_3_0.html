Release 2.3.0

And mga sumusunod na pagbabago ay ginawa sa release na ito:

Mga Mahahalagang pagbabago:

'lite' na bersyon ng ZAP

Para sa release na ito kami ay nagbibigay ng 'lite' na bersyon ng ZAP bilang karagdagan sa 'buo' na bersyon. Ito ay eksaktong naglalaman ng parehong core code, pero ito lamang ay nagbibilang ng mas kaunting mga default na add-on. Siyempre, maaring mong i-download lahat ng mga 'nawawala' na mga add-on mula sa ZAP marketplace para -ma-upgrade' ang 'lite' na bersyon sa 'full'.
Ang 'lite' na bersyon ay para sa mga taong bago sa seguridad na nangangailangan ng mababang inisyal na functionality na sana ay mas madaling masimulan. Ito rin ay magiging angkop para sa mga tao na naghahanap ng mas maliit na download o sa mga nais na eksaktong i-customize ang anumang add-ons na na-install.

Suporta para sa panig-kliyente (browser) na mga pangyayari

Maari mo nang tingnan, maharang, manipulahin, ipadalang muli at magpalabo ng mga pangyayari sa panig-kliyente. Ito ay kinabibilangan ng postMessages, kaya maari mo ng tuklasin ang DOM batay sa XSS na mga kahinaan sa postMessages. Ito ang unang yugto ng isang serye ng planadong mga pagbabago upang suportahan ang pagsusuri ng mga AJAX at HTML5 application nang mas epektibo.

Pinahusay na suporta ng pagpapatunay

Ang suporta ng ZAP sa pagpapatunay ay lubos na na naayos para madaling mapangasiwaan ang mga kumplikadong uri ng paraan at mga sitwasyon ng pagpapatunay. Naidagdag din ang suporta para sa mga itinakdang script na magpapahintulot sa iyo na pangasiwaan ang mga pamamaraan ng pasadyang pagpapatunay. Bukod pa rito, ang ZAP ay nag-uunawa na ngayon at nagpapahintulot sa iyo ng magkumpigura ng mga gumagamit ng mga aplikasyon ng web upang maisagawa ang iba't-ibang mga aksyon sa buong ZAP mula sa pananaw ng mga itinakdang manggagamit. Para makapagsimula, tingnan ang bagong Pagpapatunay at Mga Gumagamit na mga panel sa Katangian ng Sesyon para sa bawat isa sa mga itinukoy na konteksto.

Suporta para sa mga di-karaniwang app

Ang release na ito ay kinabibilangan ng suporta para sa mga aplikasyon ng 'iisang pahina' at di-karaniwang key-value na panghiwalay. Maari mo nang kontrolin ang mga setting na ito gamit ang bagong Istraktura na panel sa Katangian ng Sesyon.

Bagong Input Vectors kabilang ang mga itinakdang script

Ang ZAP ay sumusuporta ng mga bagong opsyon para sa pagtukoy ng input vectors halimbawa ang mga elemento ng isang kahilingan na aatakihin ng ZAP. Ang mga bagong opsyon ay magagamit sa Aktibong Scan Input Vectors panel ng mga opsyon. Ang suporta ay idinagdag para sa pagtukoy ng mga pasadyang script na nagtutukoy ng bagong input vectors.

Patakaran ng pag-scan - pinong grained kontrol

Ang patakaran ng pag-scan ngayon ay may isang pinong grained kontrol, nagpapahintulot sa iyo upang baguhin ang indibidwal na mga patakaran ng scanner. Maari mo ring tukuyin, i-load at i-save ang mga patakaran ng pag-scan, nagpapahintulot sa iyo na mapanatili ang isang hanay ng mga patakaran na maiging gumagawa sa iba't-ibang sitwasyon.
Bukod pa rito, sa pamamagitan ng default, ang ZAP ay hindi na ngayon mag-scan ng mga kilalang parameter ng serbisyo (e.g. __VIEWSTATE) nagpapabilis ng kabuuang proseso ng pag-scan. Ito ay ganap na maaring ikumpigura ng mga gumagamit, na nagpapahintulot sa iyo sa eksaktong tukuyin ang mga parameter na dapat balewalain ng ZAP.

Diyalogo ng Advanced na Aktibong Pag-scan

Ang bagong 'Advanced na Aktibong Pag-scan' na diyalogo ay nagpapahintulot sa iyo na eksaktong tukuyin kung paano mo gustong gumana ang aktibong scanner. Ito ay nagpapahintulot sa iyo na tukuyin ang ‘custom vectors’ na pinagtitiwalaan na tukuyin ang anumang mga string na gusto mo atakihin. Ito rin ay sumusuporta ng opsyon na mag-scan ng anumang natukoy na mga gumagamit para sa aplikasyon sa ilalim ng pagsusuri. Simulan ang isang Advanced na Aktibong Pag-scan sa pamamagitan ng Mga Kagamitan na menu o sa pamamagitan ng Atake seksyon ng right click popup menu.

Mga opsyon ng pinalawak na command line

Maari mo nang patakbuhin ang ZAP nang 'inline' halimbawa nang hindi sinisimulan ang ZAP UI o ang daemon. Sa paraang ito, maaring kang magpatakbo ng mga simpleng atake o magpatakbo ng mga script na maaring mag-access sa lahat ng ZAP functionality. Maari mo na ring i-override ang anumang opsyon na tinukoy sa file ng kumpigurasyon sa pamamagitan ng command line parameters.

Mas marami pang suporta ng API

Ang API ay pinalawak upang suportahan ang mas higit pang ZAP functionality.

File ng pina-internasyonal na tulong

Ang file ng tulong ay pina-internasyonal at nasa proseso ng pagiging isinalin sa maraming mga wika sa pamamagitan ng https://crowdin.net/project/owasp-zap-help. Kung gumagamit ka ng ZAP sa isa sa maraming wika na sinuportahan namin, ang mga file ng tulong ay kinabibilangan ng lahat ng magagamit na pagsasalin para sa wikang iyon habang binabalik ang default sa Ingles para sa mga parirala na hindi pa naisalin.
Ang mga wika na may makabuluhang halaga ng isinalin na mga pahina ng tulong ay kinabibilangan ng:
  • Bosnian
  • Pranses
  • Wikang Hapon
  • Espanyol

Mga shortcut ng keyboard

Lahat ng item sa menu ay maaring nang matawag sa pamamagitan ng keyboard shortcut. Tinukoy ang mga default para sa halot lahat ng mga kaso, ngunit maaring mo itong ma-configure sa iyong sariling kagustuhan sa mga Keyboard panel ng mga opsyon.

Bagong mga opsyon ng UI

May isang bagong opsyon para baguhin ang display upang ang napiling tab tumatagal ng hanggang sa buong screen. Ito ay kapaki-pakinabang kapag ginagamit ang ZAP sa maliit na screens. Mayroon ring opsyon para mag-toggle ng pagkakakita ng mga pangalan ng tab sa isang off para lalong makatipid ng espasyo.

Karamihan sa mga listahan ng UI ay pinapalitn ng tables, kung saan nagpapahintulot sa iyong palitan ang lapad ng kulumna at eksaktong tukuyn ang anumang ipanapakitang kulumna, at paano nagkasunod-sunod ang mga table.

Karagdagang functionality na inilipat sa add-ons

Higit pa sa mga pangunahing functionality ay nailipat sa add-ons na nagpapahintulot sa atin na makapaghatid ng mga update sa pamamagitan ng ZAP Marketplace sa halip na mangangailangan ng mga bagong buong release.
Ito ay kinabibilangan ng mga pangkat ng wika, kaya ang pagsasalin na ginawa sa ZAP UI sa pamamagitan ng https://crowdin.net/project/owasp-zap ay maaring i-download sa loob ng ZAP o kahit na awtomatikong i-install.

Bago at mas pinabuting aktibo at pasibong patakaran ng pagsusuri

Marami sa mga kalidad ng release na aktibo at pasibong patakaran ng pagsusuri ay napabuti. Mayroong bagong alpha at beta na mga patakaran ng kalidad at maraming patakaran ang naitaguyod mula sa alpha hanggang beta at mula beta hanggang sa kalidad ng release.
