Release 1.0.0

Ang mga sumusunod ng pagbabago ay para sa release na ito:

Importanteng pagbabago:

Mga Help screen

Mga help screen ay naidagdag para sa lahat ng mga functionality na ibinibigay ng ZAP.
Ang sensitibong tulong sa konteksto ay naipatupad din upang kapag pinindot mo ang F1 key, ang help screen na nauugnay sa tab na iyong pinili ay ipapakita.

Mga Break Point

Ang pagsasakatuparan ng Break Points ay nabago upang maging mas malapit sa naibigay ng software Integrated Development Environments.
Ang kontrol ng break point ay nailipat sa bagong Top Level Toolbar.
Ang mga Break point ay maaring ring i-set sa pamamagitan ng pag-right click ng mga node sa Mga Site at Mga tab ng History.

Menu ng Report

Ang Menu ng Report ay may importanteng pagbabago at ito ngayon sa kasali sa mga sumusunod na mga item ng menu:
    Gawan ng Report...Na maaring makagawa ng mga kailanganing report
    I-export ang mga Mensahe sa File...Ilipat mula sa File Menu
    I-export mga mga Tugon sa File...Ilipat mula sa File Menu
    I-export ang lahat ng mga URL sa File...Bagong functionality
    Ikompara sa ibang sesyon...Bagong functionality

Pinabuting paghahanap ng kasaysayan

Ang bagong Search tab nagpapahintulot sa iyo na maghanap ng mga regular na expression sa lahat ng mga URL, mga kahilingan at mga tugon.
Ang susunod at naunang mga button ay nagpapahintulot sa iyo na tingnan ang nahanap na mga termino sa paghahanap sa Hiling at Mga tugon na tab.

Filter ng Kasaysayan

Ang toolbar ng filter ng kasaysayan ay naidagdag na nagbibigay-daan sa iyo upang paghigpitan ang ipinapakita na mga kahilingan.
Ang mga hiling ay maaring i-filter base sa mga HTTP method, mga HTTP result codes at ibang mga tag, mga alerto, o mga tala na nai-apply na.

Mga Tala

Maaring mong iugnay ang mga tala sa anumang hiling.

Karagdagang mga Tag

Maaring mo iugnay ang mga tag sa anumang mga hiling.

Higit pang kontrol sa mga alerto

Ang Add Alert na dialog ay nagbibigay-daan sa iyo upang mano-manong magdagdag o magbago ng isang alerto.

Bagong encode/decoder

Ang bagong Encode/Decode/Hash na dialog ay pinalitan ang nakaraang Encoder/Decoder.

Passive scanning

Ang Passive scanning ay naidagdag.
Sa release na ito, ang ZAP lamang ang gagamit ng passive scanning para awtomatikong maidagdag ang mga tag.
Sa mga release sa hinaharap, ito ay maaring mag-alsa ng mga alert para sa mga potensyal na mga isyu.

Maliit na pagbabago:

Ang Top Level Toolbar

Ang Top Level Toolbar ay naidagdag na nakapagbibigay ng mga button para sa mga pangkaraniwang ginamit na functionality.

Mga buod ng Alert footer

Ang mga bilang ng High, Medium, Low at Informational na mga alert ay makikita ngayon sa footer.

Baagong itsura

Ang bagong itsura na 'Nimbus' ang ginamit.

Pamamahala ng Proxy password

Ang screen na Options Connection ay may isang bagong pagpipilian na 'Prompt for proxy credentials on start up'.
Kung napili ang pagpipiliang ito, ang iyong password ng proxy ay hindi maiimbak sa hard disk.

Ipadalang muli mula sa Sites na tab

Ang Ang Sites tab ay may bagong item sa pag-right click ng item mula sa menu na nagpapahintulot sa iyo na maipadalang muli ang kahilingan pagkatapos gumawa ng anumang mga pagbabago na gusto mo.

Nanatiling user directory sa pagitan ng mga sesyon

Ang huling directory na ginamit ng user ay naiimbak ngayon sa config file para ito ay mapanatili sa pagitan ng mga sesyon.
