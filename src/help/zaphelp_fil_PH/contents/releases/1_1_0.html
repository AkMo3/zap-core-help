Release 1.1.0

Ang mga sumusunod na pagbabago ay ginawa para sa release na ito:

Mga Mahahalagang pagbabago:

Rebranding ng OWASP

Ang ZAP ay tinanggap biliang isang OWASP na proyekto.
Ang homepage nito ngayon ay: https://www.owasp.org/index.php/OWASP_Zed_Attack_Proxy_Project

Brute Force

Ang kakayahanang mag-brute force ng mga file at mga directory batay sa code mula sa OWASP DirBuster na proyekto.
Ang bagong Brute Force tab ay nagpapakita ng mga file at mga directory na natagpuan.

Scan ng Port

Ang kakayahang mag-scan ng port sa mga site.
Ang bagong Scan ng Port na tab ay nagpapakita ng mga port na natagpuan.

Aktibong Scan tab

Ang bagong Aktiboong Scan tab ay nagpapakita ng mga request at mga response bilang resulta ng mga aktibong pag-scan ng site.

Spider tab

Ang Spider tab ngayon ay nagpapahintulot sa iyong ipagpatuloy ang paggamit ng ZAP habang nag-spider ng isang site.
Maaring mo ring itigil at ituloy ang spider.

Suporta ng Smartcard

Ang suporta ng Smart Card ay naidagdag ng proyekto ng Andiparos.
Ang sumusunod na aparato ng smartcard ay nasubok na sa Windows:
    SafeguardAladdin eToken
    Aladdin eToken Pro
    OmnikeyOmnikey 3121
    CardMan 6121
    GemaltoReflex 20 V2
    Swiss Stick
Ang sumusunod na mga aparato ng smartcard ay inulat na gumana:
    OmnikeyCardMan 4040

Attack menu

Ang bagong Sites tab i-right click ang 'Attack' menu ay nagpapahintulot sa iyo na simulan ang iba't-ibang mga scan.

Higit pang internationalisation

Lahat ng mga pangunahing mga tab at mga item ng menu ay na-internationalise na.

Lokalisasyon

Ang suporta para sa sumusunod nga mga wika ay naisagawa sa bersyong ito:
    InglesAng default na wika
    Brazilian Portuguese
    Aleman
    Polish
    Espanyol

Pagpipilian ng Wika

Sa simula ay papipiliin ka ng wikang gagamitin.
Ang mga bagong wika ay awtomatikong natuklas ng presensiya ng mga file na may mga pangalan ng form na Messages_<locale>.properties sa ZAP directory.

Maliit na mga pagbabago:

Hindi pinagana na mga 'default file' plugin

Ang mga plugin na nagtutuklas ng default na mga file ay epektibong ginawang paulit-ulit ng bagong scanner ng brute force.
Ang mga ito ay hindi pinagana.

Mga buod ng scanner

Ang mga bilang ng mga numero at mga uri ng kasalukuyang scan ay ipinapakita na ngayon sa footer.
