Release 1.2.0

Ang mga sumusunod na pagbabago ay ginawa para sa release na ito:

Mga Mahahalagang pagbabago:

Memory Leaks

Ang Memory Leaks ay naayos sa aktibong scanner at spider.

Pagtawag ng mga application

Ang mga external na mga application ay maari nang matawag mula sa mga Site at History na mga tab.

Passive Scanner

Ang passive scanner ay naghahanap na ngayon ng mga kahinaan, gaya ng:
    Mga form na autocomplete na may password na field
    Cookies na wala ang 'HttpOnly' flag
    SSL Cookies na wala ang 'secure' flag
    Mahinang pagpapatunay

Maliit na mga pagbabago:

Mga XML Report

Ang bagong 'Bumuo ng XML Report...' na menu item ay kasama na ngayon sa pinakamataas na antas ng Reports menu.

Manu-manong Request Editor at Resend dialogs

Ang mga request na naisumite ng Manu-manong Request Editor at Resend dialogs ay makikita na ngayon sa Sites at History na mga tab.
Isang bagong 'Method' na pull down ay nagpapahintulot sa iyo na palitan sa pagitan ng mga HTTP method, ito ay awtomatikong naglilipat ng maga parameter sa pagitan ng URL at ng body kapag ang POST method ay napili o hindi napili.

UI ng mga Alerto

Ang Sites tab ay nagpapakita na ngayon ng anumang alerto bilang flag sa kanan ng anumang mga pangalan ng node.
Ang bilang nga alerto sa footer ay nagpapakita na ngayon ng bilang ng magkaibang uri ng mga alerto sa halip na kabuuang bilang ng mga instance.

Pagpipilian na antala ng aktibong scanner

Ang antala sa milliseconds sa pagitan ng bawat aktibong scanner request ay maitatakda gamit ang Screen na Options Active Scan. Ito ay magdaragdag ng oras na kailangan ng isang aktibong scan ngunit mababawasan ang load sa target.

Mga pagbabago sa UI

Ang Sites tab ay kumukuha ng lahat ng kaliwang bahagi - ito ay maaring baguhin pabalik sa pamamagitan ng Options Display screen kung kinakailangan.

Ang 'toolbar' sa Request, Response at Break tabs at ang Manu-manong Request Editor at Resend dialogs ay nasa taas na ngayon sa halip na nasa ibaba.

Ang bilang ng scanner sa footer ay makikita na ngayon sa kaliwang bahagi.
