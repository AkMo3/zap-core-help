Pagpapakwala 2.4.1

Ang Pagpapakwalang ito ay may mga importanteng seguridad na inayos - ang mga gumagamit ay mahigpit na inirekomenda upang ma-upgrade agad.

Isa sa mga pagbabago na nangangahulugan na ang susi sa API ay nalikha sa pamamagitan ng default, ang kahulugan nito ay ang kahit anong aplikasyon na gumagamit ng ZAP API ay nagkamali maliban na lang kung sila ay mag-update sa pagamit ng ganyang susi.
Ang Susi sa API ay pwedeng mahanap sa Mga Opsyon na skrin sa API
Pwede mo din itakda ito mula sa ipinaguutos na linya sa pamamagitan ng opsyon katulad: 

-config api. susi=baguhin-ako-9203935709

Para sa maraming detalye tingnan ang https://github.com/zaproxy/zaproxy/wiki/FAQapikey
