API

Ang ZAP ay nagbibigay ng isang Aplikasyong Programang Interface (API) na kung saan ay pinapayagan ka na makipag-ugnayan kasama ng programatikong ZAP.

Ang API ay makukuha sa JSON, HTML at XML na mga format.
Ang isang simpleng web UI ay pinapayagan ka na magsiyasat at gamitin ang API na makukuha sa pamamagitan ng URL http://zap/ habang ikaw ay kumakatawan sa pamamagitan ng ZAP, o sa pamamagitan ng host at port ng ZAP ay nakikinig sa, hal http://localhost:8080/.

Sa pamamagitan ng default lamang ang makina ng ZAP na tumatakbo ay kayang i-access ang API. Maaari mong payagan ang ibang mga makina, na kayang gumamit ng ZAP nilang isang kinatawan, i-access sa API. Ang API ay kinumpigura gamit ang Mga pagpipilian ng API sa iskrin.

Ang API ay nagbibigay ng access sa lahat ng pinaka buod ng mga tampok ng ZAP tulad ng Aktibong pag pagsusuri at ispayder.
Ang mga bersyon sa hinaharap ng ZAP ay magpapataas ng pag-andar na magagamit sa pamamagitan ng API.
