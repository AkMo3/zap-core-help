Mga URL na Hindi Kasali sa Pangkalahatan

Paalala: ang tampok na ito ay beta.

Mga URL na Hindi Kasali sa Pangkalahatan ay isang hanay ng Regular Expressions (regex) na kumpletong binabalewala ng ZAP sa buong aplikasyon. Ang mga URL na ito ay hindi magpapakita sa Proxy, Scanner, o Spider para sa ZAP. At saka, ang URL regexs ay hindi masi-save sa global na kumpigurasyon para sa ZAP at hindi sa loob ng file ng sesyon. Ang mga URL na ito ay magpupumilit sa pagitan ng bawat paggamit ng ZAP.

Ang regex para sa mga URL ay isinaayos gamit ang Screen ng Global na Pagbukod ng URL sa mga opsyon ng ZAP.

Tingnan din

     Pangkalahatang-ideya ng UIpara sa pangkalahatang-ideya ng interface ng gumagamit
     Mga Tampokna ibinigay ng ZAP
