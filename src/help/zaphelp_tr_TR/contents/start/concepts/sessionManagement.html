Oturum Yönetimi

ZAP, çoklu oturum yönetimi tiplerini ( Oturum olarak anılır) işlemektedir Yönetim Yöntemleri ) web siteleri / webapps için kullanılabilir. Her Bağlamı , bir Oturum Yönetimi Oturumların nasıl tutulacağını belirleyen yöntem.

Şimdiye kadar, sadece çerez tabanlı ve HTTP kimlik doğrulama oturum yönetimi yöntemleri uygulanmış, ancak sistem yeni yöntemlerin kolay eklenmesini destekliyor, kullanıcı ihtiyaçlarına göre.

Çerez Temelli Oturum Yönetimi

Bu yöntem söz konusu olduğunda, oturum aşağıdaki yollarla izlenmektedir: kurabiye. Şu anda kullanılan oturum simgeleri şu adlardan içe aktarıldı: HTTP Oturumu Uzantısı.

HTTP Kimlik Doğrulama Oturum Yönetimi

Bu yöntem söz konusu olduğunda, oturum HTTP isteği başlığı Yetkilendirme ile yönetilir.
