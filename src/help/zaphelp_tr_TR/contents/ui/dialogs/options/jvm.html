JVM Seçenekleri Ekranı

Bu ekran, ZAP başlatılırken kullanılan JVM seçeneklerini yapılandırmanıza izin verir.

JVM Seçenekleri

Zap.sh veya zap.bat üzerinden ZAP çağrıldığında Java komut satırı çağrısına eklenecek serbest metin alanı.

Seçenek eklendi, böylece maksimum Java belleği ayırma havuzu boyutu ayarlanabilir, şu biçimdeki: - Xmxn nerede n bitler olarak uzunluk belirttiği.
Bu alan için güzel iyi değerler şöyle olabilirdi:

  • -Xmx256m
  • -Xmx512m
  • Xmx1024m

Diğer ZAP seçeneklerinden farklı olarak bunlar, kullanılan OS'ye bağlı olan kullanıcının varsayılan ZAP dizinindeki ".ZAP_JVM.properties" dosyasında tutulur:

  • Windows 7/8: C:\Users\<username>\OWASP ZAP
  • Windows XP: C:\Documents and Settings\<username>\OWASP ZAP
  • Linux: ~/.ZAP
  • Mac OS: ~/Library/Application Support/ZAP
Eğer bu ayarları kurarken bir hata yaparsanız, ZAP başlamayı başaramayabilir.
Eğer bu olursa, dosyayı silmek sıkıntıyı çözecektir.

ZAP'i Windows exe aracılığıyla başlatırken bu seçeneklerin kullanılmadığını unutmayın.
