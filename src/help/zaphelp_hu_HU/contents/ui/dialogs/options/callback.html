Options Callback Address screen

A Visszahívási Cím képernyőn beállíthatja, hogy a cím kimutatására használt biztonsági rések, amelyek lehetővé teszik, hogy a támadó hívás távoli Url-eket.
A korábbi verziókban a ZAP API-t használták erre a célra, de a 2.6.0-től külön végpont úgy alkalmazzák, hogy a cél rendszerek többé nem kell a hozzáférést az API.

Local Address (e.g. 0.0.0.0)

The local address that ZAP will listen on for incoming connections. The default value of '0.0.0.0' means that ZAP will listen on all available local addresses.

Remote Address

The address that will be specified in the relevant attacks. This address must be accessible from the target system. You can use the Test URL to check that this is the case.

Random Port

By default ZAP will use a different port each time it is run. If you need to use the same port (for example to allow access through firewalls) then uncheck this option.

Specify Port

Ha a Véletlenszerű Port opció nincs bejelölve, akkor ez a port, hogy ZAP hallgat. Lehet más a más portok ZAP használ, például a portot használja a proxy kapcsolat.

Test URL

Ez a teszt URL-t, hogy próbálja meg elérni a távoli rendszerek. All accesses to the Test URL will be recorded in the ZAP log file at INFO level. If you are using the ZAP UI then accesses will also be shown in the Output tab.
