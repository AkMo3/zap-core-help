Konfigurisanje Proxya

You will need to configure your browser to use ZAP as a proxy.
By default ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options Local Proxy screen. Instructions for the latest versions of the most commonly used browsers:

Chrome (na Windowsima)

    Pritisni 'Customise and control Google Chrome' button (top right)
    Select the 'Options..' menu item
    Pritisni 'Change proxy settings' button
    Pritisni 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

Firefox

    Select the 'Tools' menu
    Select the 'Options..' menu item
    Pritisni 'Advanced' button
    Select the 'Network' tab
    Pritisni 'Settings...' button
    Select the 'Manual proxy configuration' radio button
    Enter in the 'HTTP Proxy:' field the 'Address' you configured in the Options Local Proxy screen
    Enter in the 'Port' field to the right of the 'HTTP Proxy' field the 'Port' you configured in the Options Local Proxy screen
    Pritisni Connection Setting 'OK' button
    Pritisni Options 'OK' button

Firefox (on OS X)

    Select the 'Firefox' menu
    Select the 'Preferences...' menu item
    Pritisni 'Advanced' button
    Follow the instructions above from the 'Advanced' button

Internet Explorer 7 and 8

    Select the 'Tools' menu
    Select the 'Internet Options' menu item
    Select the 'Connections' tab
    Pritisni 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

Opera

    Select the 'Menu' in the top left hand corner
    Select the 'Settings' menu item
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Select the 'Network' item in the left hand list
    Pritisni 'Proxy Servers' button
    Select the 'HTTP' checkbox
    Enter in the field to the right of 'HTTP' the 'Address' you configured in the Options Local Proxy screen
    Enter in the HTTP 'Port' field the 'Port' you configured in the Options Local Proxy screen
    Pritisni Proxy servers 'OK' button
    Pritisni Preferences 'OK' button

Safari (on OS X)

    Pritisni 'Safari settings' button in the top right hand corner
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Pritisni 'Change Settings' button next to the 'Proxies' label
    Follow the 'OS X System Proxy settings' section below

Safari (on Windows)

    Pritisni 'Display a menu of general Safari settings' button in the top right hand corner
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Pritisni 'Change Settings' button next to the 'Proxies' label
    Pritisni 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

OS X System Proxy settings

    Select the 'Web Proxy (HTTP)
    Enter in the 'Address:' field the 'Address' you configured in the Options Local Proxy screen
    Enter in the 'Port' field the 'Port' you configured in the Options Local Proxy screen
    Pritisni Proxies Setting 'OK' button
    Pritisni Network Options 'Apply' button

Windows LAN Settings

    Select the 'Use a proxy server for your LAN' checkbox
    Enter in the 'Address:' field the 'Address' you configured in the Options Local Proxy screen
    Enter in the 'Port' field the 'Port' you configured in the Options Local Proxy screen
    Pritisni Local Area Network (LAN) Setting 'OK' button
    Pritisni Internet Options 'OK' button
