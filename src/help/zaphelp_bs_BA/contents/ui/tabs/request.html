Zahtjevi tab

The Request tab shows you the data sent by your browser for the request that you have highlighted in either the Sajtovi taba ili Historija tab .
The top panel shows the request header and the bottom panel shows any data also sent, for example as a result of a POST request.
A pull down is provided which allows you to see the data:
     Raw Viewas supplied
     Tabular Viewin a table with one row per attribute and value
     Hex Viewin a table showing the hex representation of all of the characters in the body

Desni klik meni

Desni klik na nod će prikazati meni koji će vam dopustiti da:

Traži...

This will bring up the Traži dijalog.

Kodiraj/Dekodiraj/Heš...

This will bring up the Encode/Decode/Hash dialog.
If you have highlighted any text then this will be automatically included in the dialog.

Kopiraj

This will copy the selected string to the clipboard.
