1.2.0版本

本版本进行了以下更改：

重大变化：

内存泄漏

内存泄漏已经固定在 主动扫描器 和 spider.

调用应用程序

现在可以从“站点”和“历史记录”选项卡调用外部应用程序。

被动扫描器

被动 扫描器 现在可以查找漏洞，例如：
    具有密码字段的自动填充表单
    没有“HttpOnly”标志的Cookie
    没有“安全”标志的SSL Cookies
    弱认证

小更改：

XML 报告

一个新的“生成XML报告...”菜单项现在包含在顶层工具栏中 报告 菜单

手动请求编辑器和重新发送对话框

由 手动请求编辑器 和 重发送 对话框提交的请求现在显示在 站点 和 历史 标签中。
一个新的下拉“方法”允许您在HTTP方法之间切换，这会在URL和主体之间自动移动参数当POST方法被选择或取消选择时。

警报UI

站 点 选项卡现在将任何警报显示为任何节点名称右侧的标志。
警报计数在 页脚 ，现在显示的是如何提供不同类型的警报而不是实例总数。

主动扫描延迟选项

每个主动扫描器之间的延迟以毫秒为单位 （active scanner） request can now be set via the Options Active Scan screen这将增加主动扫描的时间但会减轻目标的负担。

UI变化

The 站点 tab now takes up all of the left hand side - this can be changed back via the Options Display screen if required.

The 'toolbar' on the 请求, 响应 and 中断 tabs and the 手动请求编辑器 and 重发送 dialogs is now at the top rather than the bottom.

The scanner counts in the footer are now displayed on the right hand side.
