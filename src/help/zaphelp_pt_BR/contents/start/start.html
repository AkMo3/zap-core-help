Iniciando

A maneira mais rápida de iniciar o uso do ZAP é usar o complemento Quick Start, que é instalado por padrão.
Isso permite que você insira uma URL onde o ZAP irá passar o spider e depois fazer varredura ativa.
Para um teste mais profundo, você deve explorar seu aplicativo usando o seu navegador ou testes automatizados de regressão, fazendo proxy através do ZAP.

At its heart ZAP is a man-in-the-middle proxy.
To get the most out of ZAP you need to configure your browser or functional tests to connect to the web application you wish to test through ZAP.
If required you can also configure ZAP to connect through another proxy - this is often necessary in a corporate environment.

You can quickly and easily launch browsers that are pre-configured to proxy through ZAP via the Quick Start tab. This can launch any of the most common browsers that you have installed with new profiles.
If you would like to use any of your browsers with an existing profile, for example with other browser add-ons installed, then you will need to manually configure your browser to proxy via ZAP.
If you know how to set up proxies in your web browser then go ahead and give it a go!
If you are unsure then have a look at the Configuring proxies section.

Uma vez que você tenha configurado o ZAP como proxy do seu navegador, em seguida, tente se conectar ao aplicativo da web que você estará testando.
Se você não pode se conectar a ele então verifique suas configurações de proxy novamente. Você precisará verificar as configurações de proxy do seu browser e também as configurações de proxy do ZAP.
Também vale a pena verificar se o aplicativo que você está tentando testar está ativo!

Quando você conectar ao seu aplicativo através do seu navegador, dê uma olhada no ZAP novamente. Você então deve ver uma ou mais linhas nas guias Sites e histórico.
Se isso aconteceu, está tudo funcionando. Se não, então você precisará verificar as configurações de proxy de novo.

A próxima coisa a fazer é começar um teste de penetração básico.
