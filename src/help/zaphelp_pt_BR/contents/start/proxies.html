Configurando os proxies

You will need to configure your browser to use ZAP as a proxy.
By default ZAP uses an Address of 'localhost' and a Port of '8080', but these can be changed via the Options Local Proxies screen. Instruções para as últimas versões dos navegadores mais usados:

Chrome (no Windows)

    Pressione o Botão "Personalizar e controlar o Google Chrome" (canto superior direito)
    Selecione Item de menu 'Opções...'
    Pressione o Botão 'Alterar configurações de proxy'
    Pressione o Botão 'Configurações da LAN'
    Siga o Seção 'Configurações de LAN do Windows' abaixo

Firefox (on Windows)

    Selecione Menu 'Ferramentas'
    Selecione Item de menu 'Opções...'
    Pressione o Botão 'Avançado'
    Selecione Aba 'Rede'
    Pressione o Botão 'Configurações...'
    Selecione Botão de opção 'configuração manual de proxy'
    Digite no 'HTTP Proxy:' field the 'Address' you configured in the Options Local Proxies screen
    Digite no 'Port' field to the right of the 'HTTP Proxy' field the 'Port' you configured in the Options Local Proxies screen
    Pressione o Botão 'OK' de configuração de conexão
    Pressione o Botão 'OK'

Firefox (on Linux)

    Selecione'Edit' menu
    Selecione'Preferences' menu item
    Follow theinstructions above from the 'Advanced' button

Firefox (on OS X)

    Selecione 'Firefox' menu
    Selecione 'Preferences...' menu item
    Press the 'Advanced' button
    Follow the instructions above from the 'Advanced' button

Internet Explorer 7 and 8

    Selecione 'Tools' menu
    Selecione 'Internet Options' menu item
    Selecione 'Connections' tab
    Press the 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

Opera

    Select the 'Menu' in the top left hand corner
    Selecione 'Settings' menu item
    Selecione 'Preferences' menu item
    Select the 'Advanced' tab
    Select the 'Network' item in the left hand list
    Press the 'Proxy Servers' button
    Select the 'HTTP' checkbox
    Enter in the field to the right of 'HTTP' the 'Address' you configured in the Options Local Proxies screen
    Enter in the HTTP 'Port' field the 'Port' you configured in the Options Local Proxies screen
    Press the Proxy servers 'OK' button
    Press the Preferences 'OK' button

Safari (on OS X)

    Pressione o 'Safari settings' button in the top right hand corner
    Selecione Item de menu 'Preferências'
    Selecione Aba 'Avançado'
    Pressione o Botão 'Alterar configurações' ao lado do rótulo de 'Proxies'
    Follow the 'OS X System Proxy settings' section below

Safari (on Windows)

    Press the 'Display a menu of general Safari settings' button in the top right hand corner
    Select the 'Preferences' menu item
    Select the 'Advanced' tab
    Pressione o 'Change Settings' button next to the 'Proxies' label
    Pressione o 'LAN Settings' button
    Follow the 'Windows LAN Settings' section below

OS X System Proxy settings

    Selecione 'Web Proxy (HTTP)
    Digite no 'Address:' field the 'Address' you configured in the Options Local Proxies screen
    Digite no 'Port' field the 'Port' you configured in the Options Local Proxies screen
    Pressione o Proxies Setting 'OK' button
    Pressione o Network Options 'Apply' button

Windows LAN Settings

    Select the 'Use a proxy server for your LAN' checkbox
    Enter in the 'Address:' field the 'Address' you configured in the Options Local Proxies screen
    Enter in the 'Port' field the 'Port' you configured in the Options Local Proxies screen
    Press the Local Area Network (LAN) Setting 'OK' button
    Press the Internet Options 'OK' button
