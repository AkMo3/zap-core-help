Configurando os proxies

Você precisará configurar seu navegador para usar o ZAP como um proxy.
Por padrão o ZAP usa um endereço de 'localhost' e uma porta de '8080', mas estas podem ser alteradas através da tela de opções de Proxy Local. Instruções para as últimas versões dos navegadores mais usados:

Chrome (no Windows)

    Pressione o Botão "Personalizar e controlar o Google Chrome" (canto superior direito)
    Selecione Item de menu 'Opções...'
    Pressione o Botão 'Alterar configurações de proxy'
    Pressione o Botão 'Configurações da LAN'
    Siga o Seção 'Configurações de LAN do Windows' abaixo

Firefox

    Selecione Menu 'Ferramentas'
    Selecione Item de menu 'Opções...'
    Pressione o Botão 'Avançado'
    Selecione Aba 'Rede'
    Pressione o Botão 'Configurações...'
    Selecione Botão de opção 'configuração manual de proxy'
    Digite no Campo ' HTTP Proxy:' o 'endereço' configurado na tela de opções de Proxy Local
    Digite no Campo 'Porta', à direita do campo 'HTTP Proxy', a 'porta' configurada na tela de opções de Proxy Local
    Pressione o Botão 'OK' de configuração de conexão
    Pressione o Botão 'OK'

Firefox (no OS X)

    Selecione Menu 'Firefox'
    Selecione Item de menu 'Preferências...'
    Pressione o Botão 'Avançado'
    Siga as instruções acima do botão 'Avançado'

Internet Explorer 7 e 8

    Selecione Menu 'Ferramentas'
    Selecione Item de menu 'Opções da Internet'
    Selecione Aba 'Conexões'
    Pressione o Botão 'Configurações da LAN'
    Siga o Seção 'Configurações de LAN do Windows' abaixo

Ópera

    Selecione 'Menu' no canto superior esquerdo
    Selecione Item de menu 'Configurações'
    Selecione Item de menu 'Preferências'
    Selecione Aba 'Avançado'
    Selecione Item 'Rede' na lista da esquerda
    Pressione o Botão 'Servidores de proxy'
    Selecione Caixa de seleção 'HTTP'
    Digite no campo à direita de 'HTTP' o 'endereço' configurado na tela de opções de Proxy Local
    Digite no Campo 'Porta' HTTP o 'endereço' configurado na tela de opções de Proxy Local
    Pressione o Botão de 'OK' de servidores proxy
    Pressione o Botão 'OK' Preferências

Safari (no Mac OS X)

    Pressione o Botão 'Configurações do safari' no canto superior direito
    Selecione Item de menu 'Preferências'
    Selecione Aba 'Avançado'
    Pressione o Botão 'Alterar configurações' ao lado do rótulo de 'Proxies'
    Siga a Seção de 'Configurações de Proxy do sistema OS X' abaixo

Safari (no Windows)

    Pressione o Botão 'Exibir um menu de configurações gerais do Safari' no canto superior direito
    Selecione Item de menu 'Preferências'
    Selecione Aba 'Avançado'
    Pressione o Botão 'Alterar configurações' ao lado do rótulo de 'Proxies'
    Pressione o Botão 'Configurações da LAN'
    Siga a Seção 'Configurações de LAN do Windows' abaixo

Configurações de Proxy do sistema OS X

    Selecione ' Web Proxy (HTTP)
    Digite no Campo ' HTTP Proxy:' o 'endereço' configurado na tela de opções de Proxy Local
    Digite no Campo 'Porta' HTTP o 'endereço' configurado na tela de opções de Proxy Local
    Pressione o Botão 'OK' de Definição de Proxies
    Pressione o Botão 'Aplicar' opções de rede

Configurações do Windows LAN

    Selecione Check box 'Utilizar um servidor proxy para a LAN'
    Digite no Campo "endereço" o mesmo configurado na tela de opções de Proxy Local
    Digite no Campo 'Porta' HTTP o 'endereço' configurado na tela de opções de Proxy Local
    Pressione o Botão 'OK' na definição rede local (LAN)
    Pressione o Botão 'OK' em Opções da Internet
