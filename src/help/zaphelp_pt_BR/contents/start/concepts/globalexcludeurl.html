URLs globalmente excluídas

Nota: este recurso é beta.

URLs globalmente excluídas são um conjunto de expressões regulares (regex) que o ZAP ignora completamente em todo o aplicativo. Essas URLs não aparecerão no Proxy, Scanner ou Spider para o ZAP. Além disso, os regexs de URLs serão salvos na configuração global do ZAP e não dentro do arquivo de sessão. Essas URLs serão mantidas entre cada uso do ZAP.

As regex para URLs são configuradas usando a Tela Global de exclusão de URLs nas opções ZAP.
