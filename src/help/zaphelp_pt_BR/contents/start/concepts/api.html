API

O ZAP fornece uma API (Application Programming Interface) que permite a você interagir programaticamente com ele.

A API está disponível em formatos JSON, HTML e XML.
Uma interface web simples permite que você explore e use a API. Ela está disponível através da URL http://zap/ quando você estiver fazendo proxy através do ZAP, ou através do host e porta ZAP que estiver escutando o ZAP. Por exemplo, http://localhost:8080/ /.

By default only the machine ZAP is running on is able to access the API. You can allow other machines, that are able to use ZAP as a proxy, access to the API. The API is configured using the Options API screen.

A API fornece acesso a recursos centrais do ZAP tais como varredura ativa e spider.
Versões futuras do ZAP vão aumentar a funcionalidade disponível através da APi.
