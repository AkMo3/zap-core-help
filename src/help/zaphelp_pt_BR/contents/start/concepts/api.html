API

O ZAP fornece uma Application Programming Interface (API) que permite a você interagir programaticamente com o ZAP.

A API está disponível em formatos JSON, HTML e XML.
A documentação da API está disponível através da URL http://zap/ quando você em proxy via ZAP.

Note-se que há segurança incorporada à API, e por isso ela está desativada por default.
Se ativada, em seguida, a API está disponível para todas as máquinas que são capazes de usar o ZAP como proxy.
Por padrão o ZAP escuta somente em 'localhost' e assim a API só pode ser usada na máquina host.

A API fornece acesso a recursos centrais do ZAP tais como varredura ativa e spider.
Versões futuras do ZAP vão aumentar a funcionalidade disponível através da APi.

A API é configurada usando a tela de opções de API.
