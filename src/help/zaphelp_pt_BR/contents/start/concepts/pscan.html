Varredura Passiva

O ZAP faz varredura passiva de todas as respostas do aplicativos da web que está sendo testado.
A Varredura passiva não altera as respostas de forma alguma e, portanto, é segura para uso.
A varredura é executada em um thread de segundo plano, para garantir que ele não retarde a exploração de um aplicativo.

Nesta versão a varredura passiva é usada para automaticamente adicionar tags e criar alertas para potenciais problemas.

Um conjunto de regras para a marcação automática é fornecido por padrão.
Elas podem ser alteradas, excluídas ou adicionadas ao através da tela de opções de Tags de varredura passiva.

Os alertas gerados por varredura passiva podem ser configurados usando-se a tela de opções de regras de varredura passiva.
