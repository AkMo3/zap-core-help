Tokens anti CSRF

Anti CSRF tokens são parâmetros aleatórios de (pseudo) usados para proteger contra ataques de Cross Site Request falsificação (CSRF).
No entanto eles também fazem um trabalho de testadores de penetração mais difícil, especialmente se os tokens são regenerados sempre que um formulário é solicitado.

O ZAP detecta tokens anti CSRF pelos nomes de atributo - a lista de nomes de atributos considerados tokens anti CSRF é configurada usando-se a tela de opções Anti CSRF.
Quando o ZAP detecta esses tokens, grava o valor deles e qual URL gerou o token.
A varredura ativa e o difusor têm opções que fazem o ZAP automaticamente regenerar os tokens se necessário.
