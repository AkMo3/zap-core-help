Modos

ZAP has a 'mode' which can be:

  • Seguro - nenhuma operação potencialmente perigosa é permitidas
  • Protegido - você só pode executar ações perigosas (potencialmente) em URLs dentro do escopo
  • Padrão - como nas versões anteriores, você pode fazer qualquer coisa
  • ATAQUE - novos nós que estão no escopo são ativamente varridos tão logo são descobertos
É recomendável que você use o modo protegido para garantir que só ataque sites que você escolha.

O modo pode ser alterado através da barra de ferramentas (or the ZAP API) and is persisted between sessions.

Examples of the things that will not be possible in either Safe mode or in Protected mode when not acting on URLs in the Scope:
  • Spidering
  • Varredura ativa
  • Fuzzing/difusão
  • Navegação forçada
  • Interrupção (na interceptação)
  • Reenviar solicitações/requests
You can define the Políticas de varredura to be used for the Attack mode the Tela de opções da Varredura Ativa.
