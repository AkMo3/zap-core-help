Barra de Ferramentas Superior

Esta barra de ferramentas fornece um conjunto de controles para as funcionalidades mais usadas.

Modo suspenso

Isso permite que você altere o modo atual.

Nova sessão

este botão é o mesmo que o item de menu do menu arquivo 'Nova sessão'.

Sessão aberta

este botão é o mesmo que o item de menu de 'Sessão aberta' do menu arquivo.

Persistir sessão...

Este botão é o mesmo que o item de menu do menu arquivo 'Persistir Sessão...'.

Instantâneo de Sessão

este botão é o mesmo que no item de menu de 'Instantâneo de Sessão' do menu arquivo.

Propriedades da sessão...

Este botão é o mesmo que o item de menu de 'Propriedades' do menu arquivo.

Opções...

Este botão é o mesmo de Opções de Ferramentas.

Mostrar todas as guias

Este botão revela todas as guias ocultas.

Ocultar abas sem alfinete

este botão oculta todas as guias não estão com 'alfinete '. Guias podem ser fixadas e liberadas por meio dos ícones de 'alfinete' que são mostrados quando a aba é selecionada.

Mostrar ícones e nomes de abas

Este botão alterna a exibição dos nomes de abas.

Expandir aba de Sites

isso altera a exibição para que a janela de 'árvore' que contém a guia Sites estende-se por toda a extensão do lado esquerdo.
Isto reduzirá a quantidade de espaço disponível para a janela de 'informações'.

Expandir informações das abas

Isso altera a exibição para que a janela de 'informações' ocupe por toda a largura da parte inferior.
Isto reduzirá o espaço disponível para a janela de 'árvore'.

  Full Layout

This changes the display so that the selected tab takes up the full screen.
This is useful when using ZAP on small screens.

Guias de request e resposta ao lado

isso altera a exibição para que as guias de solicitação e resposta fiquem lado a lado.
Isto aumenta as informações que podem ser exibidas, mas significa que você não pode ver a solicitação e a resposta ao mesmo tempo.

Request mostrado acima resposta

Isso altera a exibição para que o painel de request seja mostrado acima do painel de resposta.
Isso diminui as informações que podem ser exibidas, mas significa que você pode ver a solicitação e a resposta ao mesmo tempo.

Painéis de solicitação e resposta ao lado

isso altera a exibição para que o painel de pedido seja mostrado à esquerda do painel de resposta.
Isso diminui as informações que podem ser exibidas, mas significa que você pode ver a solicitação e a resposta ao mesmo tempo.

/ Ligar / Desligar interrupção em todas as solicitações e respostas

Isso Liga e Desliga umponto de interrupção'global' que vai capturar e exibir o próximo request ou resposta na Aba de Interrupções.
Você pode então alterar qualquer parte da solicitação ou resposta e enviá-las para o aplicativo de destino, pressionando qualquer dos botões 'Passo' ou 'Continuar'.
Como alternativa, você pode pressionar o botão 'Drop' para abandonar a solicitação ou resposta.
Você pode escolher entre botões 'combinado' ou separados para as solicitações e respostas através da tela de opções de pontos de interrupção

/ Ligar / Desligar interrupções em todos os requests

Isso liga e desliga um ponto de interrupção'global' que vai capturar e exibir a próxima solicitação/request na aba de interrupções.
Em seguida, você pode alterar qualquer parte da solicitação e enviá-la para o aplicativo de destino, pressionando qualquer dos botões 'Passo' ou 'Continuar'.
Alternativamente você pode pressionar o botão de 'Drop' para abandonar o request.
Você pode alternar entre um único botão 'combinado' oui dois separados para as solicitações e respostas, através da tela de opções de pontos de interrupção

/ Ligar / Desligar interrupção em todas as respostas

Isso liga e desliga um ponto de interrupção'global' que irá interceptar e exibir a próxima resposta na Aba de Interrupções.
Você então pode alterar qualquer parte da resposta que quiser e enviá-la para seu navegador pressionando qualquer dos botões 'Passo' ou 'Continuar'.
Alternativamente você pode pressionar o botão de 'Drop' para abandonar a request.
Você pode alternar entre um botão 'combinado' ou dois separados para as solicitações e respostas através da tela de opções de pontos de interrupção

Passo

isso permite que a request ou resposta retidos continuem seu caminho para o aplicativo ou para o seu navegador com quaisquer alterações que você tenha feito.
O ponto de interrupção'global' permanecerá definido para que a próxima solicitação ou resposta também seja capturada.
Este botão é habilitado somente quando uma solicitação ou resposta são capturados.

Continuar

o ponto de interrupção'global' será desligado para que as solicitações e respostas seguintes não sejam mais capturadas pelo ZAP - a menos que você tenha definido pontos de interrupção em URLs específicas.
Este botão é habilitado somente quando uma solicitação ou resposta é capturada.

Drop

isso descarta solicitação ou resposta retida para que não seja passada para o aplicativo ou seu navegador.
Este botão é habilitado somente quando uma solicitação ou resposta é capturada.

Adicionar um ponto de interrupção HTTP personalizado

isso exibe a caixa de diálogo Adicionar ponto de interrupção que permite a você especificar os critérios para um ponto de interrupção.

Gerenciar Complementos

isto mostra a caixa de diálogo Gerenciar Complementos, permitindo que você descubra, instale e atualize complementos do 'marketplace' do ZAP.
Ele também permite que você desinstale complementos.

/ Ligar/desligar o modo Usuário Forçado

isto alterna entre Usuário Forçado ligado e desligado
O botão é habilitado apenas quando você tiver definido um usuário forçado para pelo menos um contexto, o que pode ser feito através do diálogo de Contextos de sessão.

Observe que Complementos pode inserir botões adicionais.
