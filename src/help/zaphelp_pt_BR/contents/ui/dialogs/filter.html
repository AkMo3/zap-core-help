Filtro

Isso permite que você defina filtros que são aplicados para solicitações e respostas.

Os seguintes filtros são suportados por padrão.

Modificar o user agent para outros navegadores

Detectar conteúdo inseguro ou potencialmente malicioso em respostas HTTP

Detectar e alertar tentativas de 'Set-cookie' em respostas HTTP por modificação

Burlar cache do navegador (Ignorar IfModifiedSince)

Fazer log dos cookies enviados pelo navegador

Fazer log de queries GET únicas em arquivo

Fazer log de queries POST únicas em arquivo

Fazer log de requests e respostas em arquivo

Substituir o corpo das requisições HTTP utilizando um padrão definido

Substituir o cabeçalho das requisições HTTP utilizando um padrão definido

Substituir o corpo das respostas HTTP utilizando um padrão definido

Substituir o cabeçalho das respostas HTTP utilizando um padrão definido

Enviar ID de requisição de sessão

Este filtro irá adicionar uma tag especial de cabeçalho para cada solicitação enviada ao servidor. Assim você pode rastrear as solicitações do ZAP, ao investigar sua própria aplicação web. É útil ter uma identificação distinta para várias solicitações, ao analisar, por exemplo, arquivos de log do servidor HTTP.

O cabeçalho de solicitação é modificado como abaixo (exemplo):

X-ZAP-RequestID: –
  • sessionName é o nome da sessão codificado para URL
  • number começa em um; é incrementado em cada request; é válido somente na sessão
