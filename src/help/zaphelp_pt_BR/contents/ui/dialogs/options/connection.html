Tela das opções de conexão

A tela das opções de conexão permite a você configurar as opções de conexão do ZAP:

Timeout em segundos

Isso facilita o teste em aplicações lentas.

Default User Agent

The user agent that ZAP should use when creating HTTP messages (for example, spider messages or CONNECT requests to outgoing proxy).

Single Cookie Request Header

Controls whether the ZAP's managed cookies ((Habilitiar o Rastreamento da Sessão - Cookie) should be set on a single "Cookie" request header or multiple "Cookie" request headers, when sending an HTTP request to the server.

DNS

TTL das queries bem-sucedidas

Defines for how long the successful DNS queries should be cached:
  • Número Negativo, cache para sempre;
  • Zero, desativa o cache;
  • Número positivo, o número de segundos que as consultas serão armazenadas.
Nota: Changes are applied after a restart.

A opção também pode ser definida usando o -config como argumento de linha de comando com a chave connection.dnsTtlSuccessfulQueries.

Security Protocols

Allows to choose the SSL/TLS versions enabled for outgoing connections (for example, to servers). At least one version must be enabled, versions unsupported by the JRE will be unselected and disabled.
The option SSLv2Hello must be selected in conjunction with at least one SSL/TLS version.

Use Proxy chain

This section allows you to connect to another proxy for outgoing connections.
This is often required in a corporate environment.

Proxy authentication

This section allows you to configure the outgoing proxy authentication.
