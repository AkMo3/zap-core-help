Tela das opções de conexão

A tela das opções de conexão permite a você configurar as opções de conexão do ZAP:

Timeout em segundos

Isso facilita o teste em aplicações lentas.

Header de requisição de um cookie

Define se os cookies gerenciados pelo ZAP(Habilitiar o Rastreamento da Sessão - Cookie) devem ser definidos num único header de requisição de cookie ou em múltiplos headers de requisição, toda vez que for enviada uma requisição HTTP ao servidor.

Usar cadeia de proxies

Esta aseção permite a você conectar a outro proxy para as conexões de saída.
Isto é geralmente solicitado em ambientes corporativos.

Autenticação via Proxy

Esta seção permite a você configurar a autenticação do proxy de saída.
