Tela de Entrada de opções para Vetores de Varredura Ativa

Esta tela permite que você configure a entrada de vetores para varredura ativa .
Estes são os elementos que a varredura ativa irá atacar.
Varrer todos os elementos disponíveis vai demorar mais, mas não varrer alguns deixará escapar algumas vulnerabilidades.

Alvos injetáveis

O elemento de requisição que a varredura ativa atingirá:
    String de query da URLPar de valores-chave na URL query, ou seja, depois do '?'
    Dados do POSTPar de valores-chave nos dados da requisição POST
    Caminho da URLElementos de caminho (path) na URL de requisição, ou seja, os elementos separados por '/'
    Headers de HTTPHeaders de HTTP da requisição
    Dados do cookieRequest de cookies

Janelas de entrada de vetor

Os formatos de dados que a varredura ativa atingirá:
    Dados multipart de formulários
    atributos/tag XML
    JSON
    Google Web Toolkit
    id/filtro OData

Habilitar entrada de vetores em scripts

Se esta opção estiver selecionada, a varredura ativa utilizará quaisquer vetores em script habilitados.
Vetores de entrada em scripts são scripts que você pode escrever ou importar para o ZAP e permitem que você atinja elementos que não são suportados por default.

Esta tela também permite a você configurar os parâmetros a serem ignorados pela varredura ativa.
