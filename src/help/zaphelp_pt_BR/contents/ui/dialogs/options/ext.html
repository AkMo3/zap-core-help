Tela das opções de Extensões

Esta tela permite a você habilitar/desabilitar extensões disponíveis no ZAP.

Habilitar/desabilitar Extensões

Extensões habilitadas serão carregadas pelo ZAP, ampliando assim as funcionalidades que elas proporcionam. As extensões desabilitadas não serão carregadas.

Extensões do núcleo

Extensões do núcleo são aquelas que se forem desabilitadas impedirão o ZAP de ter um mínimo de funcionalidade. As extensões marcadas como de núcleo não podem ser desabilitadas.

Nota: extensões do núcleo podem ser desabilitadas pelo arquivo de configuração, embora isso não seja recomendável.

Dependências das extensões

Algumas extensões podem depender de outras para funcionar. Quando uma dessas extensões (dependências) estiver desabilitada, fará com que o ZAP automaticamente desabilite todas as extensões que dependem dela. Então pode acontecer que ao desabilitar uma extensão várias outras (que dependem dela) também ficarão desabilitadas. Extensões que foram automaticamente desabilitadas não poderão ser habilitadas até que as dependências estejam habilitadas.

Localização das extensões disponíveis no ZAP

Para tornar uma extensão disponível para o ZAP ela precisa estar num dos seguintes locais:
  • diretório "plugin", localizado na pasta do ZAP;
  • diretório "plugin", localizado na pasta do usuário do ZAP.

Nota: você precisa reiniciar o ZAP para que as mudanças tenham efeito.
