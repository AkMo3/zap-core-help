Tela de Opções do Banco de Dados

Esta tela permite a você configurar as opções de banco de dados:

Compactar (na saída)

Permite a compactação do database quando o ZAP é encerrado; compactar o database garante uso mínimo de espaço em disco mas também vai fazer com que o encerramento do ZAP demore mais.

Tamanho máximo do corpo da requisição

É o maior tamanho do corpo em bytes que o ZAP irá permitir.

Maior tamanho do corpo da Resposta

É o maior tamanho do corpo em bytes que o ZAP irá permitir.

Entrada de opções de persistência de nova sessão

Se estiver ligado o diálogo Persistir na Sessão vai aparecer quando uma nova sessão for criada.

Opção default

A opção default para novas sessões - se a entrada definida acima estiver habilitada, então a opção default será selecionada.
Caso contrário esta será a ação a ser iniciada.
