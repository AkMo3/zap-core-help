Tela das Opções para verificar Atualizações

Esta tela permite que você configure se o ZAP verificará se você está executando a versão mais recente quando começa, e em caso contrário o que deve fazer.

Verificar atualizações ao iniciar

Se selecionado, então o ZAP irá automaticamente verificar se há atualizações para o aplicativo inteiro e para todos os add-ons quando iniciar.
É altamente recomendável que você ligue esta caixa.
Se por algum motivo que você escolher não, deverá verificar manualmente e com frequência as atualizações.
ZAP vai fazer apenas um pedido de atualização e a única informação incluída será a versão atual em que ele está.

Baixar automaticamente novas versões do ZAP

Se selecionado, então o ZAP irá automaticamente baixar novos releases ZAP quando eles estiverem disponíveis e pedir para você instalá-los.

Instalar automaticamente as atualizações dos Add-ons que você tem

Se selecionado, o ZAP fará automaticamente o download e instalará quaisquer updates nos add-ons que você tiver no ZAP.

Instalar automaticamente as atualizações para as regras de scanner que você instalou

Se selecionado, o ZAP fará automaticamente o download e instalará quaisquer updates para as regras de varredura que você adotou.
Estes são um subset do conjunto completo de Add-ons instalados.

Informar novos add-ons em qualidade de release

Se selecionado, o ZAP irá informá-lo se e quando qualquer add-on for promovidos para 'qualidade de release'.
Estes add-ons serão exaustivamente testados e revisados, e você pode ter certeza que eles são da mais alta qualidade.

Informar novos add-ons em qualidade beta

Se selecionado, o ZAP irá informá-lo se e quando qualquer add-on for promovidos para 'qualidade beta'.
Estes add-ons terão sido testados e revisados, e serão considerados de razoável qualidade e a maioria cumpre sua finalidade.
No entanto, eles podem estar incompletos ou precisar de mais testes.
Alguns dos add-ons incluídos no ZAP por padrão estão ainda no nível beta.

Informar novos add-ons em qualidade alfa

Se selecionados, o ZAP irá informá-lo se e quando novos Complementos estarão adicionados na qualidade 'alpha'.
Estescomplememntos estarão revisados, mas geralmente estão na fase inicial de desenvolvimento.
Podem estar incompletos, conter problemas relevantes ou causar problemas de estabilidade.

Diretórios de complementos

ZAP will load all of the add-ons from the default plugin directories.
You can also add as many additional add-on directories as you need by adding them here.
This can be very useful when running multiple ZAP instances on one machine, for example in a CI environment.
You can then just download new or updated add-ons once and immediately share them between multiple instances.
Add-ons downloaded into shared directories will be only added to new ZAP instances, not to ones that are already running, with the exception of the instance that actually downloaded the add-on.

Diretório de Download

This is the directory ZAP will download new or updated add-ons to.
By default it will be the local directory, but you can also choose one of the shared directories as long as this ZAP instance can write to it.
ZAP will only move add-ons to a shared directory after they are completely downloaded, so that other ZAP instances do not try to load an incomplete add-on.
