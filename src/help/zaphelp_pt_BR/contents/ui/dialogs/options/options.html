Diálogos de opções

A caixa de diálogo de opções permite que você configure o ZAP.
Ele inclui as seguintes telas:
    Varredura Ativa
    Entrada de vetores da varredura ativa
    Tokens anti CSRF
    API
    Pontos de interrupção
    Certificado
    Verificar se há Atualizações
    Conexão
    Banco de dados
    Exibir
    Certificados SSL Dinâmicos
    Teclado
    Proxy Local
    Regras de varredura passiva
    Tags de varredura passiva
    Scripts
    Busca
    Spider
