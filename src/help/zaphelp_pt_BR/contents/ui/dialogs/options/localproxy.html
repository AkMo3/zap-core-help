Tela de opções de Proxy Local

A tela de opções de conexão permite que você configure o endereço e a porta na qual o ZAP aceita conexões de entrada.
É este endereço e a porta que você deve configurar no seu navegador para usar como proxy.

Modificar/remover o cabeçalho de solicitação "Accept-Encoding"

Permite ao proxy modificar ou remover o campo de cabeçalho de solicitação "Accept-Encoding", de modo que nenhuma transformação de codificação é feita para a resposta.
Esta opção deve ser sempre habilitada, a menos seja um teste das transformações de codificação.
As mensagens codificadas não serão corretamente examinadas (seja por varredura passiva ou ativa).
