Tela de opções da API

Esta tela permite que você configure as opções da API:

Habilitado

Se habilitado, a API estará disponível para todas as máquinas que possam usar o ZAP como proxy.

Somente em segurança

Se habilitado, então a API só estará disponível via HTTPS. Caso contrário estará disponível tanto em HTTP quanto em HTTPS.

Chave API

É preciso fornecer uma chave para todas as 'ações' da API e também para algumas das 'outras' operações.
A chave de API é usada para impedir que sites mal-intencionados acessem a API do ZAP.
É altamente recomendado que você estabeleça uma chave, a não ser que esteja utilizando o ZAP num ambiente completamente isolado.

Desabilitar a chave de API

Selecionar esta opção desativa a chave de API.
Isso não é recomendado a menos que você estiver usando o ZAP em um ambiente completamente isolado, porque permite que sites mal-intencionados acessem a API do ZAP.

Informar detalhes de erros através da API

Se esta opção for selecionada, então, mais detalhes do erro são retornados via API.
Isso não é recomendado, exceto para fins de depuração, pois essas mensagens de erro podem vazar informações para sites maliciosos.
Observe que os detalhes completos do erro sempre são gravados no arquivo de log do ZAP.

Preenchimento automático da chave de API na interface de usuário da API

Se esta opção estiver selecionada então a chave de API é incluída automaticamente na interface do usuário da API.
Isso não é recomendado a menos que você estiver usando o ZAP em um ambiente completamente isolado, porque permite que sites mal-intencionados acessem a chave de API do ZAP.

Habilitar JSONP

A seleção dessa opção habilita o formato JSONP.
Isso pode ser útil para algumas aplicações, mas geralmente não é recomendado pois aumenta a área de superfície de ataque do ZAP, ou seja, características que um site mal-intencionado pode explorar.
Se o JSONP estiver ativado todas as operações de API usando JSONP (incluindo exibições) exigirá a chave de API para impedir que sites mal-intencionados acessem informações sigilosas mantidas pelo ZAP, tais como as chaves de sessão.
