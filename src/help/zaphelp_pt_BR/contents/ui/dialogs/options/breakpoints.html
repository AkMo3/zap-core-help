Tela de opções dos pontos de interrupção

Esta tela permite a você configurar as opções dos pontos de interrupção:

Confirmar derrubar mensagem presa

Pede confirmação quando um request ou resposta são derrubados.

ZAP sempre no topo quando atinge um ponto de interrupção

Se estiver selecionado, o ZAP fica no topo de todas as janelas sempre que um ponto de interrupção aparece.

Modo botões de Break

Isto controla se haverá um botão de break simples/combinado na Barra de ferramentas superior ou se serão separados para requests e respostas.
