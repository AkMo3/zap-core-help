Codificar / Decodificar / Hash

Isto permite codificar, decodificar fazer hash em texto.

Campos comuns

A caixa de diálogo tem um campos que é comum a todas as abas:

Texto para Codificar/Decodificar/Hash

Este campo é para o texto que você deseja codificar, decodificar ou fazer hash.
Se algum texto estiver selecionado quando a caixa de diálogo é chamada, então ele será colocado neste campo. Os outros campos serão atualizados dinamicamente se você alterar o texto

Aba dos campos Codificar

Esta aba possui os seguintes campos:

Codificação Base 64

Este campo só de leitura será atualizado com a codificação base 64 do texto que você digitou.

URL Codificada

Este campo só de leitura será atualizado com a codificação em URL do texto que você digitou.

Codificação ASCII Hex

Este campo só de leitura será atualizado com a codificação ASCII Hex do texto que você digitou.

Abas dos campos Decodificar

Esta aba possui os seguintes campos:

Decodificação Base 64

Este campo só de leitura será atualizado com a decodificação base 64 do texto que você digitou.
Se houver não decodificações válidas então o campo ficará desativado.

URL Decodificada

Este campo só de leitura será atualizado com a decodificação URL do texto que você digitou.

Decodificação ASCII Hex

Este campo só de leitura será atualizado com a decodificação ASCII Hex do texto que você digitou.
Se houver não decodificações válidas então o campo ficará desativado.

Aba dos campos de Hash

Esta aba possui os seguintes campos:

Hash SHA1

Este campo só de leitura será atualizado com o Hash SHA1 do texto que você digitou.

Hash MD5

Este campo só de leitura será atualizado com o Hash MD5 do texto que você digitou.
