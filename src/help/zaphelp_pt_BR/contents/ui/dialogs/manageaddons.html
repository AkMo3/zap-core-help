Gerenciar complementos

Isso permite a você gerenciar os Complementos que instalou e baixar novos complementos do Marketplace.

Duas abas estão disponíveis:

Instalado

Esta aba mostra a versão do ZAP que está rodando, todos os complementos instalados e permite a você verificar se há atualizações para o ZAP ou para qualquer dos complementos.
Passar o cursor sobre um complemento faz com que mais informações sobre ele sejam exibidas.
Você também pode desinstalar complementos nesta aba - normalmente complementos são removidos do interface do usuário ZAP tão logo você os desinstale.

Marketplace

Esta aba mostra todos os complementos que você não instalou e que estão disponíveis no Marketplace de Complementos.
Você precisará 'verificar atualizações' para poder ver os complementos disponíveis.
Passar o cursor sobre um complemento faz com que mais informações sobre ele sejam exibidas.
Você pode instalar qualquer um dos complementos listados - normalmente os Complementos são adicionados à interface do usuário do ZAP assim que você fizer sua instalação.

Acessado via

     Barra de Ferramentas SuperiorBotão 'Gerenciar Complementos'
