HTTP Sessions tab

Esta aba mostra a você o conjunto de sessões HTTP identificadas para cada Site, conforme detectado pela extensão de sessões HTTP.

O Site ao qual a as informações se referem pode ser selecionado pela barra de ferramentas ou pela Aba de Sites.

A barra de ferramentas fornece um botão ("nova sessão"), que permite a você iniciar uma nova sessão, forçando todas as requests a ficarem sem o conjunto de tokens de sessão, de modo que o servidor considera que há uma nova sessão. Isto permite a criação de uma nova sessão sem destruir a anterior.

Cada uma das entradas na tabela de sessões inicialmente tem um nome, mas ele pode ser alterado selecionando-se a célula 'Nome' e editando-a.

Cada uma das entradas na tabela de sessões pode ser examinada com clique no botão direito, o que ativa um menu pop-up com as seguintes opções:

  • Remover Sessão - deleta a sessão
  • Definir como ativa (disponível somente em sessões não-ativas) - marca esta sessão como ativa. Se qualquer sessão foi anteriormente definida como ativa, isso será revertido; se ela não tiver qualquer valor de token, será deletada.
  • Desmarcar como ativa (disponível apenas na sessão ativa) - marca esta sessão como não sendo mais ativa. Se a sessão não tiver quaisquer valores de token, ela será deletada.
Em relação à sessão activa, mais detalhes podem ser lidos na página de ajuda de conceitos gerais da Extensão de sessões HTTP

Para cada sessão, você pode ver:
  • Ativa - se a sessão está ativa ou não
  • Name - o nome de sessão
  • Valores dos Tokens de sessão - a lista de valores associados a cada um dos tokens de sessão. As entradas são separadas pelo símbolo ';' (ponto-e-vírgula).
  • Mensagens Combinadas - o número de mensagens HTTP combinadas com esta sessão por causa de sua extensão
