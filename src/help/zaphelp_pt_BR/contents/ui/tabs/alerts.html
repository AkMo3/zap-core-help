Aba de alertas

A guia alertas mostrar os alertas que surgiram durante a sessão. Os alertas são exibidos em uma árvore, em ordem de risco, no painel da esquerda, e o número total de alertas é mostrado embaixo de cada nó da árvore.

Selecionar um alerta com um clique irá exibir no painel da direita. Duplo clique num alerta exibirá a caixa de diálogo Adicionar alerta que permitirá que você altere os detalhes do alerta.

Filtragem de alertas

Os botões de filtro, exibidos imediatamente acima da árvore, permitem restringir os alertas a serem mostrados:

 /    Mostrar apenas URLs do Escopo / Mostrar todas as URLs

Permite mostrar apenas os alertas de URLs que estão no escopo.

 / Linkar / Deslinkar com seleção de Sites

Permite mostrar apenas os alertas do nó selecionado na Aba de Sites .

Nota: Selecionar um filtro irá desativar os outros, porque eles são mutuamente exclusivos.

  Edit Alert

Will display the Add Alert dialog which will allow you to change the alert details, for the selected alert.

Note: If multiple alerts are selected in the tree, the most recently selected item is edited.

Menu do botão direito

Right clicking on a node will bring up a menu which will allow you to:

Resend...

This will bring up the Resend dialog which allows you to resend the request associated with the alert after making any changes to it that you want to.
