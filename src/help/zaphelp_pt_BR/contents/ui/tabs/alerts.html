Aba de alertas

The Alerts tab show the Alerts that have been raised in this session. The alerts are displayed in a tree in risk order in the left hand pane, and each node of the tree shows the total number of alerts underneath it.

Selecting an alert with one click will display it in the right hand pane. Double clicking an alert will display the Add Alert dialog which will allow you to change the alert details.

Filtragem de alertas

Os botões de filtro, exibidos imediatamente acima da árvore, permitem restringir os alertas a serem mostrados:

 /    Mostrar apenas URLs do Escopo / Mostrar todas as URLs

Permite mostrar apenas os alertas de URLs que estão no escopo.

 / Linkar / Deslinkar com seleção de Sites

Permite mostrar apenas os alertas do nó selecionado na Aba de Sites .

Nota: Selecionar um filtro irá desativar os outros, porque eles são mutuamente exclusivos.

Menu do botão direito

Clicar com o botão direito em um nó faz aparecer um menu que lhe permitirá:

Reenviar...

Isso irá abrir o Diálogo de reenvio que lhe permite reenviar a solicitação associada a um alerta, podendo fazer nela quaisquer alterações que você deseje.
