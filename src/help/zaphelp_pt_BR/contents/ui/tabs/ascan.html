Aba de Varredura Ativa

A aba de Varredura Ativa permite a você executar uma Varredura Ativa (Active Scan) .

O botão 'Nova Varredura' abre o diálogo de Varredura Ativa, que permite a você especificar exatamente o que deve ser escaneado.

A barra de ferramentas contém um conjunto de botões que permitem iniciar, parar, pausar e continuar a varredura selecionada.
Uma barra de progresso mostra o quanto progrediu a verificação do site selecionado.
O valor 'Varreduras em andamento' mostra quantas varreduras estão ativas - passar o cursor sobre esse valor irá mostrar num pop-uo uma lista dos sites que estão sendo examinados.
The 'Show scan progress details' button launches the Scan Progress dialog which allows you to see details about which rules are running, skip individual rules and see a chart of the responses.

Menu do botão direito

Right clicking on a node will bring up a menu which will allow you to:

Excluir de

This menu has the following submenus:

Proxy

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
This can be used to ignore URLs that you know are not relevant to the system you are currently testing.
Os nós podem ser incluídos novamente através do diálogo Propriedades da Sessão .

Scanner

This will prevent the selected nodes from being actively scanned.
Os nós podem ser incluídos novamente através do diálogo Propriedades da Sessão .

Spider

This will prevent the selected nodes from being spidered.
The nodes can be included again via the Session Properties dialog

Reenviar...

This will bring up the Resend dialog which allows you to resend the request after making any changes to it that you want to.

Novo Alerta...

This will bring up the Add Alert dialog which allows you to manually record a new alert against this request.

Mostrar na Aba de Sites

This will show the selected message in the Sites tab.

Ver no Navegador

This will open the URL of the selected node in your default browser.
