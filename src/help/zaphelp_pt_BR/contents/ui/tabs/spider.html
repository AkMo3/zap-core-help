Aba Spider

A aba Spider mostra o conjunto de URIs únicas encontradas pelo Spider durante as varreduras.

O botão 'Nova Varredura' lança o diálogo do Spider, para permitir que você especifique exatamente o que deve ser escaneado.
O Spider pode ser executado em vários Sites ao mesmo tempo e os resultados de cada verificação são mostrados selecionando-se a varredura através do menu suspenso 'Progresso'.

A barra de ferramentas fornece um conjunto de botões que permitem iniciar, parar, pausar e continuar a verificação. Uma barra de 'Progresso' mostra o quanto a varredura selecionada através do menu suspenso 'Progresso' já progrediu.


Para cada solicitação, você pode ver:
  • Processdo - se a URI foi processada pelo Spider ou foi ignorada dessa busca por causa de uma regra (por exemplo, estar fora do escopo)
  • Método - o método HTTP, por exemplo GET ou POST, através do qual o recurso deve ser acessado
  • URI - o recurso encontrado
  • Flags - qualquer informação sobre a URI (por exemplo, se é uma semente/seed ou porque não foi processada)
