Pantalla de opciones de configuración de la regla

Esta pantalla te permite configurar el comportamiento de reglas de exploración específicas, tanto pasivas cómo activas.

Seleccione cualquiera de las filas enumeradas para ver los detalles de la configuración específica y para cambiar el valor asociado. Todas las reglas, incluyendo las individuales, pueden ser reiniciadas y ajustadas a su valor original.

Las reglas incorporadas incluyen:

NombrePor defectoDescripción
rules.common.sleep15La duración de tiempo en segundos usada para sincronizar ataques.
rules-cookie.ignorelistLista de nombres de las cookies separada por una coma. Las cookies incluidas en ésta lista serán ignoradas cuando se realice el escaneo de problemas relacionados a las cookies.
rules.csrf.ignorelistUna lista de identificadores separados con una coma. Cualquier Forma con un nombre o una identificación que coincida con uno de estos identificadores será ignorado cuando se realice el escaneo de los tokens Anti-CSRF faltantes. Solo use esta función para ignorar las Formas que ya sabe que sean seguras, como por ejemplo, las formas de busqueda.
