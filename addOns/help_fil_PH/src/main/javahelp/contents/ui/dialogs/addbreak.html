Magdagdag ng dialog ng Break Point

Pinapayagan ka nito na magtakda ng isang break point sa isang URL.

Ang break point ay tinukoy sa pamamagitan ng isang ekspresyon ng regex. <0 /> Kung bibisitahin mo ang isang URL na tumutugma dito pagpapahayag pagkatapos ay maaapektuhan ito ng ZAP at pahihintulutan kang baguhin ang kahilingan at / o ang tugon.

Ma-access sa pamamagitan ng

     Ang kasaysayan na tab'Break...' right click menu item
     Ang Sites tab'Break...' right click menu item

Tignan din

     Kabuuan ng UIpara sa kabuoan gumagamit ng interface
     Mga dayalogopara sa mga detalye ng diyalogo o mga popup
