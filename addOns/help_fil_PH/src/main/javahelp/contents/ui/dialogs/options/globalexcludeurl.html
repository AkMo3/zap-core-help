Mga Opsyon ng Screen Pagbukod ng URL nang Global

Paalala: mga tampok na ito ay beta.

Ang screen na ito ay nagpapahintulot sa iyo na magkumpigura ng Pagbukod ng URL nang Global na regexs sa ZAP:

Regex

Ang regex ng URL na binabalewala ng ZAP. I-click ang mga button na Idadag, Baguhin, o Tanggalin para ma-edit ang bawat regex.

Pinagana

Kung naka-check ang kahon, ang regex URL na ito ay babalewalain ng ZAP.
