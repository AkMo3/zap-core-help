Mga Opsyon ng JVM screen

Ang screen na ito ay nagpapahintulot sa iyo na magkumpigura ng mga opsyon ng JVM na ginagamit sa pagsisimula ng ZAP.

Mga Opsyon ng JVM

Isang libreng format na patlang ng teksto na ididagdag sa pagtawag ng Java command line kapag nagtatawag sa ZAP sa pamamagitan ng alinman sa zap.sh o zap.bat.

Ang opsyon ay naidagdag upang ang laki ng pool ng pinakamataas na alokasyon ng memory ng Java ay maaaring itakda, na kung saan ay nasa pormang: -Xmxn kung saan ang n ay ang laki sa bytes.
Magandang halaga para sa patlang na ito ay maaaring:

  • -Xmx256m
  • -Xmx512m
  • -Xmx1024m

Hindi katulad ng ibang mga opsyon ng ZAP, ang mga ito ay gaganapin sa file na ".ZAP_JVM.properties" sa mga default ZAP directory ng gumagamit, na nakasalalay sa ginagamit na OS:

  • Windows 7/8: C:\Users\<username>\OWASP ZAP
  • Windows XP: C:\Documents and Settings\<username>\OWASP ZAP
  • Linux: ~/.ZAP
  • Mac OS: ~/Library/Application Support/ZAP
Kung may nagawa kang kamalian kapag nagtatakda ng mga opsyon na ito, ang ZAP ay maaaring mabigong makapagsimula.
Kapag ito ay nangyari, ang pagtanggal sa file na ito ay dapat aayos sa problema.

Alalahanin na ang mga opsyon ay hindi ginagamit kapag nagsisimula sa ZAP sa pamamagitan ng Windows exe.
