Magsakodigo / mabasa / pamamagitan ng Hash diyalogo

Ito ay nagpapahintulot sa iyo na Magsakodigo, mabasa o pamamagitan ng hash teksto.

Karaniwang mga bukid

The dialog has one field that is common to all of the tabs:

Teksto ay naka-encode/decoded/hashed

Patlang na ito ay para sa mga teksto na gusto mong maging naka-encode, decoded o hashed.
Kung anumang teksto ay pinili nang mahihingi ang mga diyalogo ay ilagay ito sa patlang na ito. Ang iba pang mga patlang ay ia-update dynamic kung mong baguhin ang teksto

Magsakodigo ng mga bukid ng tab

Tab na ito ay ang mga sumusunod na larangan:

Magsakodigo ng base 64

Ito basahin lamang field ay ia-update na may mga baseng 64 encoding ng teksto ipasok mo.

Magsakodigo ng URL

Ang nabasa lamang na patlang ay maa-update sa pag-encode ng URL ng teksto na iyong ipinasok.

ASCII Hex Encode

Ang nabasa lamang na patlang ay maa-update sa pag-encode ASCII hex ng teksto na iyong ipinasok.

I-decode ang mga patlang ng tab

Ang tab na ito ay may mga sumusunod na field:

Base 64 Decode

Ang nabasa lamang na patlang ay maa-update sa base 64 na pag-decode ng teksto na iyong ipinasok.
Kung walang wastong pag-decode pagkatapos ay mai-disable ang field.

URL Decode

Ang nabasa lamang na field na ito ay maa-update gamit ang URL decoding ng teksto na iyong ipinasok.

ASCII Hex Decode

Ang nabasa lamang na patlang ay maa-update gamit ang ASCII hex decoding ng teksto na iyong ipinasok.
Kung walang wastong pag-decode pagkatapos ay mai-disable ang field.

Mga patlang ng tab ng Hash

Ang tab na ito ay may mga sumusunod na field:

SHA1 Hash

Ang nabasa lamang na patlang ay maa-update sa SHA1 hash ng teksto na iyong ipinasok.

573/5000 MD5 Hash

Ang nabasa lamang na patlang ay maa-update gamit ang MD5 hash ng teksto na iyong ipinasok.
