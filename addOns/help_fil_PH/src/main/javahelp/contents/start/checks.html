Mga Patakaran ng Pang-scan

Ang ZAP ay sumusuporta sa dalawang aktibo at pasibo na patakaran ng pag-scan.

Lahat ng mga patakaran ay inilalaman ng add-ons upang sila ay maaaring mabilis at madaling mabago.
Bilang defailt, ang ZAP ay ipanapadala lang ng mga patakaran ng kalidad ng 'Release', nganut maaari kang mag-install ng 'Beta' at 'Alpha' na mga patakaran ng kalidad sa pamamagitan ng Pamahalaan ang diyalogo ng mga Add-on.

Mga Patakaran ng Pag-scan ay nagtutukoy kung anong mga patakaran ang tumatakbo at paano sila ipinatakbo.
Maaari kang magkaroon ng maraming mga patakaran ng pag-scan hangga't gusto mo upang matalakay ang iba't-ibang mga sitwasyon.

Upang pamahalaan ang mga patakaran ng pang-scan, buksan ang Diyalogo ng Tagapamahala ng Patakaran ng Pag-scan
