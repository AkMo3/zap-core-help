Mga Filter

PAALALA: Ang mga filter ay may bisa na pinalitan ng Replacer add-on at mga script na mas malakas at may kakayahang umangkop.
Ang mga filter ay tatanggalin sa hinaharap na bersyon ng ZAP.

Ang mga filter ay nagdaragdag ng mga ekstrang tampok na maaaring ilapat sa bawat kahilingan at tugon.
Bilang default, wala filter ay pinagana sa una. Ang pagpagana ng lahat ng filter ay maaaring magpabagal sa proxy.

Ang mga filter ay naisaayos sa pamamagitan ng Diyalogo ng mga Filter.

Tingnan din

     Pangkalahatang-ideya ng UIpara sa pangkalahatang-ideya ng interface ng gumagamit
     Mga Tampokna ibinigay ng ZAP
