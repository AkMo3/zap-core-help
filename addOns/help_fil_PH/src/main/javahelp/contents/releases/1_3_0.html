Release 1.3.0

Ang mga sumusunod na pagbabago ay ginawa para sa release na ito:

Mga Mahahalagang pagbabago:

Fuzzing

Ang mga string sa isang response ay maari na i-fuzz para subukang hanapin ang mga kahinaan.
Ang mga Anti CRSF token ay maaring napansin at awtomatikong nagawa sa fuzzing.
Ang functionality na ito ay batay sa code mula sa OWASP JBroFuzz na proyekto.

Dynamic SSL certificates

Ang suporta para sa SSL na mga koneksyon ay pinabuti at pinadali.
Ang mga gumagamit ay maaring lumikha ng sarili nilang root certificate at ipamahagi ito sa kanilang mga HTTP client.
Tingnan sa Option: Dynamic SSL Certificates para sa karagdagang mga detalye.

Daemon mode at API

Ang pagsisimula ng ZAP na may "-daemon" command line na opsyon ay maging sanhi ng pagtakbo sa background na naka-'headless' mode, ibig sabihin na walang UI na nakikita.
Ang paunang API ay naipatupad sa XML, JSON at HTML.
Kung ang ZAP ay tumatakbo bilang isang daemon, ang API ay awtimatikong gumana, kung hindi man ay ang API ay dapat gumana sa pamamagitan ng Options API screen.
Ang API ay maaring tingnan sa pamamagitan ng pagbukas ng http://zap/ sa iyong browser kung nagsisilbing proxy sa ZAP.

Integrasyon ng Beanshell

Ang BeanShell ay isang interactive na Java shell na maaring gamitin para patakbuhin ang mga Beanshell script.
Ang mga script na ito ay pinasimpleng porma ng Java na gumamit ng maraming elemento mula sa syntax ng Java, ngunit sa mas panisimpleng scripting format.
Lahat ng Java code ay balido ding Beanshell code.
Ang integrasyon ng Beanshell sa OWASP ZAP ay nagbibigay-daan sa io na magsulat ng mga script gamit ang mga ZAP function at data set.
Ito ay maaring maging isang napakalakas na tampok para sa pagsusuri ng mga web application.

Buong internationalisation

Lahat ng nakikitang string ay lubos na ngayong na-internationalise.

Maliit na mga pagbabago:

Hex view

Ang Request, Response tabs ay nagbibigay ng opsyong 'Hex View' na magpapakita ng request at response bodies sa hex format.

Resulta ng paghahanap

Ang Tab ng paghahanap ay nagpapakita ng lahat ng mga instance sa isang string natagpuan sa halip na ang unang instance lamang sa bawat request o response.

Ibukod ang mga URL

Ang mga URL ay maaring ibukod mula sa aktibong scanner, spider at mula sa proxy.

Suporta sa Pagkopya

Karamihan sa mga panel ay nagbibigay ng 'right click' 'Kopya' na opsyon sa menu, kabilang ang Port Scan at Brute Force na mga panel.

Suporta sa Undo/Redo

Lahat ng mga input field ay sumusuporta ng Undo/Redo na mga aksyon gamit ang mga default Undo/Redo accelerator ng mga operating system:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Suporta ng port scanner proxy at timeout na opsyon

Ang Port Scanner ay maaring gamitan sa palabas na proxy (kung nakaayos) at ang timeout na naka-milliseconds ay maari ring i-set.

Mga button ng request at response break

Mayroon na ngayong 2 'Set Break' na mga button upang payagan ang mga break na ma-set sa lahat ng mga request at response.

Palawakin nga mga Site at Impormasyon na tab buttons

Mayroon na ngayong 2 buttons na nagbibigay-daan sa iyong palitan sa pagitan ng pagkakaroon ng Sites at Impormasyon na mga tabs na pinalawakan.

Break tab icon ay nagbabago ng kulay kapag na-hit ang break point

Adjustable na timeout

Ang Option: Connection screen ay nagbibigay-daan sa iyong ma-set ang timeout sa segundo para gawing mas madali ang pagsubok ng mabagal na mga application.

Mga Library update

Karamihan sa mga library na ginamit ng ZAP ay na-update sa pinakabagong bersyon.

Mga kilalang Isyu:

Mac OS X: Dynamic SSL at Google Chrome

Ang kasalukuyang Dynamic SSL ay hindi gumagana kapag ang gamit ay Google Chrome. Ito ay dahil sa isang hindi nalutas na kilalang isyu sa Google Chrome at Mac OS X Keychain. Kapag nag-aangkat ng OWASP ZAP's Root CA sa keychain at nagri-request ng isang SSL website, ang "Maling certificate" na kamaliang mensahe ay nakikita.
