Version 1.0.0

Les modifications suivantes ont été apportées dans cette version :

Changements majeurs :

Écrans d'aide

Les écrans d'aide ont été ajoutés pour toutes les fonctionnalités actuellement offertes par ZAP.
L'aide contextuelle a également été implémentée afin que lorsque vous appuyez sur la touche F1, l'écran d'aide associé à l'onglet que vous avez sélectionné s'affichera.

Points d'arrêt

L'implémentation des Points d'arrêt a été changé pour être plus proche de celle fournie par les logiciels d'Environnements de Développement Intégrés.
Les contrôles du point de rupture ont été déplacés vers la nouvelle Barre d'outils de Haut Niveau.
Les points d'arrêt maintenant peuvent également être définies en faisant un clic droit sur les nœuds dans les Sites et les Onglets d'Histoire.

Menu du rapport

Le Menu du rapport a été significativement changé et inclut maintenant les objets du menu suivants :
    Génération de rapport...Qui maintenant génère des rapports sur demande
    Exporter les messages dans un fichier...Déplacé depuis le Menu du fichier
    Exporter la réponse dans un fichier...Déplacé depuis le Menu du fichier
    Exporter TOUTES les URLs dans un fichier...Nouvelle fonctionnalité
    Comparer avec une autre session...Nouvelle fonctionnalité

Amélioration de la recherche de l'histoire

La nouvelle Barre de Recherche vous permet de rechercher des expressions régulières dans toutes les URLs, les demandes et les réponses.
Les boutons Suivant et Précédent vous permettent de voir les termes de votre recherche dans la Requête et Onglets de Réponse.

Filtre de l'histoire

La barre d'outils du filtre de l'histoire a été ajoutée et elle vous permet de limiter les demandes qui sont affichées.
Les requêtes peuvent être filtrées selon les méthodes HTTP, les codes de résultat HTTP, et tous les tags, les alertes, ou les notes qui ont été appliquées.

Notes

Il est maintenant possible d'associer les notes lors d'une requête.

Les Tags Multiples

Il est maintenant possible d'associer plusieurs tags lors d'une requête.

Plus de contrôle sur les alertes

La Boîte de Dialogue d'Alerte vous permet d'ajouter ou de changer manuellement une alerte.

Nouvel encodeur/décodeur

Une nouvelle Boîte de dialogue Encoder/Décoder/Hash a remplacé le précédent Encodeur/Décodeur.

Scan Passif

Le Scan Passif a été ajouté.
Dans cette version de ZAP n'utilisera uniquement qu'un scan passif pour ajouter automatiquement les Tags.
Dans de futures versions, cela pourra également augmenter les alertes pour des problèmes potentiels.

Changements mineurs :

La Barre d'outils de Haut Niveau

La Barre d'outils de Haut Niveau a été ajoutée et cela fournit un ensemble de boutons pour une fonctionnalité couramment utilisée.

Résumés des alertes dans le pied de page

Les Hautes, Moyennes, Basses et Informatives alertes sont maintenant affichées dans le pied de page.

Nouveau look

Un nouveau look 'Nimbus' est utilisé.

Gestion de mot de passe Proxy

L'écran des Options de Connexion a une nouvelle option pour 'Demander des informations d'identification de Proxy au démarrage'.
Si cette option est sélectionnée alors votre mot de passe Proxy ne sera pas sauvegardé dans le disque dur.

Renvoyer depuis les onglets des Sites

L' Onglet des sites a un nouvel élément de menu qui vous permet de renvoyer la requête après avoir effectué toutes les modifications que vous souhaitez en faisant un clic droit.

Répertoire de l'utilisateur maintenu entre les sessions

Le dernier répertoire utilisé par l'utilisateur est désormais stocké dans le fichier de configuration afin qu'il soit maintenu entre les sessions.
