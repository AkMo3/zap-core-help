1.0.0版本

本版本进行了以下更改：

重大变化：

帮助界面

为ZAP现在提供的所有功能添加了帮助面板。
上下文相关的帮助也已经实现，所以当您按F1键时，将显示与所选标签关联的帮助面板。

断点

断点的 实现 已经变得更接近软件集成开发环境所提供的。
断点控制已经移到新的 顶级工具栏.
断点现在也可以通过右击节点来设置 站点 和 历史记录选项卡.

报告菜单

报告 菜单 已经发生了重大的变化，现在包括了以下菜单项：
    生成报告现在按需生成报告
    将信息导出到文件中...从文件菜单移入
    导出响应信息到文件中...从文件菜单移入
    将所有的URL导出到文件新功能
    与另一个会话进行比较新功能

改进的历史搜索

新的 搜索标签 允许您在所有的URL，请求和响应中搜索正则表达式。
下一个和上一个按钮允许您查看找到的搜索项 请求 和 响应选项卡.

历史过滤器

历史过滤器工具栏 已被添加，这可以让您限制显示哪些请求。
请求可以基于HTTP方法，HTTP结果代码，和任何 标签, 警报或者 备注 （之前已被应用的）

备注

您现在可以将 备注 和任何请求相关联。

多重标签

您现在可以将多个 标签 与任何请求相关联。

更好的控制警报

添加 警报对话框 使您可以手动添加或更改警报。

新的编码器/解码器

一个新的 编码/解码/哈希会话 已经取代了之前的编码器/解码器。

被动扫描

被动扫描 已经添加。
在这次的版本中，ZAP将只使用被动扫描自动添加 标签.
在未来的版本中，它也可能会触发 警报 当遇到潜在问题的时候。

小更改：

顶级工具栏

顶级工具栏 已添加，用来提供常用功能的设置按钮。

警报页脚摘要

高，中，低不同级别的和信息的 警报 现在可以显示在 页脚.

新的外观和体验

使用新的外观。

代理密码处理

选项连接屏幕 有一个新选项，用于'启动时提示代理凭据'。
如果选择此选项，则您的代理密码将不会存储在硬盘上。

从站点选项卡重新发送

站点 选项卡 有一个新的右键菜单选项，允许您在对它进行任何您想要的更改后重新发送请求。

会话之间维护的用户目录

用户使用的最后一个目录现在被保存在配置文件中，以便在会话之间进行维护。
