编辑菜单

此菜单处理在特定选项卡中查找字符串，在所有 请求和响应中搜索字符串以及管理会话跟踪。

查找...

这将打开 查找对话框 它允许您在当前所选的窗口查找字符串。

启用会话跟踪(Cookie)

This allows session details stored in cookies to be tracked.
必须选择此选项以启动"使用当前跟踪会话"复选框 重发送 和 手动请求编辑器对话框.
会话跟踪确保所有请求都随最新的会话详情一起发送。
For example you may record a session when logged in as one user and then logout and login as another user.
在没有会话跟踪的情况下，如果您从第一个会话中重新发送请求，则它将使用第一个会话中的cookie。
在有会话跟踪的情况下，如果您重新发送相同的请求，则它将会使用第二个会话中的cookie。

重置会话状态

这将清除会话跟踪。

搜索...

This selects the 搜索标签 它允许您在所有url，请求和响应中搜索正则表达式。

下一个

This selects the next occurrence of the last string searched for.
The relevant message will be selected in the Search tab and the string will be displayed and highlighted in the 请求 或 响应 tab as appropriate.

前一个

This selects the previous occurrence of the last string searched for.
The relevant message will be selected in the Search tab and the string will be displayed and highlighted in the 请求 或 响应 tab as appropriate.

请注意，加载项可以添加其他菜单项。
