Release 2.5.0

The following changes were made in this release:

ZAP API Changes:

VIEW authorization / getAuthorizationDetectionMethod

Most visszatér adatok csomagolva egy objektum neve engedélyKimutatásiMódszer Például: 
{"authorizationDetectionMethod":{"statusCode":"-1"..."headerRegex":""}}
instead of: 
{"statusCode":"-1"..."headerRegex":""}

VIEW context / context + technologyList

Both now return data wrapped in an object called "context" For example: 
{"context":{"id":"1", ..., "inScope":"true","loggedOutPattern":""}}
instead of: 
{"id":"1", ..., "inScope":"true","loggedOutPattern":""}

ACTION spider / scan + scanAsUser

Mindkét már támogatja egy új, opcionális 'subtreeOnly' paraméter, amely korlátozza a pók, hogy a megadott részfa. Az 'url' paraméter is választható, ameddig egy érvényes 'helyi' paraméter tartozék

New 'stats' component

Az új 'statisztika' API alkatrész hozzáférést biztosít a stats most által fenntartott ZAP.

Vegye figyelembe, hogy néhány meglévő komponensek is új műveletek, lásd a API Web UI for more details.
