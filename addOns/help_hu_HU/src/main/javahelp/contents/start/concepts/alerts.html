Riasztások

Egy riasztást, hogy egy potenciális biztonsági rés, illetve a kapcsolódó külön kérésre.
Egy kérést, lehet, hogy több, mint egy figyelmeztető jelzés.

Figyelmeztető jelzések jelennek meg a UI-val egy zászló jelzi, kockázat:
      Magas
      Közepes
      Alacsony
      Informational
      False Positive

Alerts can be raised by various ZAP components, including but not limited to: active scanning, passive scanning, scripts, by addons (extensions), or manually using the Add Alert dialog (which also allows you to update or change alert details/information).

Alerts are flagged in the History tab with a flag which indicates the highest risk alert.
All alerts are listed in the Alerts tab and a count of the total number of alerts by risk is shown in the footer.

Alert overrides

Alerts raised by ZAP include both generic and specific information about the alerts raised. A konkrét információkat közvetlenül kapcsolódik a potenciális problémát találtak, például az URL-t, majd a paraméter befolyásolja. Az általános információkat tartalmaz a dolgokat, mint egy leírás, linkek a kapcsolódó online források.

Cserélje ki, vagy adjunk hozzá, hogy az általános információk segítségével egy figyelmeztető felülíró' a konfigurációs fájl. Ez lehetővé teszi, hogy azokat az adatokat tartalmazza, hogy az adott vállalat, például a megbízott politikák, belső linkeket, vagy tanácsok a sajátos technológiák használata.

Figyelmeztető felülíró konfigurációs fájl UTF-8 ingatlan tartalmazó fájlt csak az információt szeretne változtatni. Kezdődő sorokat a '#' kezelik észrevételeket figyelmen kívül hagyni.

The format is: 

<alertid>.<property> = [ + | - ] <your information>
A következő tulajdonságok támogatott:
  • name
  • description
  • solution
  • otherInfo
  • reference
For example 
# 40012 = Reflected XSS
40012.solution = Follow our company specific guidelines at http://internet.example.com/xss.html
If the value starts with a '+' then it is appended to the existing information.
If it starts with a '-' then it is prepended to the existing information.
If it does not start with a '+' or '-' then it replaces the existing information.

A riasztási felülíró konfigurációs fájl is megadható keresztül API, Options Alert screen or using the command line option: 

-config alert.overridesFilename=<filename>
