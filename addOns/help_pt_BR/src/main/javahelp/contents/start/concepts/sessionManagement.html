Gerenciamento de sessão

O ZAP trata vários tipos de gerenciamento de sessão (chamados de Métodos de gerenciamento de sessão) que podem ser usados para sites / webapps. Cada contexto tem um método de gerenciamento de sessão definido, que determina como as sessões são mantidas.

Até agora, apenas um método de gerenciamento de sessão baseado em cookie foi implementado, mas o sistema oferece suporte para fácil acréscimo de novos métodos, de acordo com as necessidades do usuário.

Gerenciamento de sessão baseado em cookies

No caso deste método, a sessão está sendo monitorada através de cookies. Atualmente, os tokens de sessão usados são importados da extensão Sessões Http.

Gerenciamento de sessão de autenticação HTTP

No caso deste método, a sessão é gerenciada com o cabeçalho de solicitação HTTP autorização.
