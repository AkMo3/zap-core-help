Fuzzing/Difusão

A difusão ou fuzzing é uma técnica que envia muitos dados inválidos ou inesperados para um alvo.

ZAP permite fuzz com qualquer request, e pode usar um conjunto de payloads pré-configurado.
Para fazer fuzzing com uma string de solicitação:

  • Selecione uma solicitação/request na aba de Sites ou histórico
  • Destaque (hilight) a seqüência de caracteres sobre a qual você deseja fazer fuzzing na aba de solicitação
  • Clique com o botão direito na guia de solicitação/request e selecione 'Fuzz...'
  • Selecione a categoria de Fuzz e um ou mais Fuzzers
  • Pressione o botão de Fuzz
  • Os resultados serão listados em seguida na aba Fuzzer; selecione-os para ver todas as requests e respostas.
Você também pode pesquisar seqüências de caracteres nos resultados de fuzz usando a Aba de Busca.

O Fuzzing é configurado usando-se a tela de opções de fuzzing. Arquivos adicionais de fuzzing podem ser adicionados através dessa tela ou podem ser colocados manualmente no diretório "fuzzers" onde foi instalado ZAP - então se tornarão disponíveis depois de reiniciar o ZAP.

Essa funcionalidade é baseada no código do projeto OWASP JBroFuzz e inclui os arquivos do projeto fuzzdb.
Note que alguns arquivos do fuzzdb foram deixados de fora já que eles fazem alguns anti-vírus sinalizarem como contendo vírus.
Você pode substituí-los (e atualizar fuzzdb) baixando a versão mais recente do fuzzdb e expandi-lo na biblioteca 'fuzzers'.
