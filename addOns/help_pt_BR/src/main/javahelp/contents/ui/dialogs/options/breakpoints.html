Tela de opções dos pontos de interrupção

Esta tela permite que você configure opções de ponto de interrupção :

Confirmar a mensagem de soltar preso

Pede confirmação quando um request ou resposta são derrubados.

Só parar em mensagens do escopo

O ponto de interrupção 'global' só será acionado por solicitações que estão no escopo.
Pontos de interrupção personalizados não serão afetados.

ZAP sempre no topo quando chega a um ponto de interrupção

Se estiver selecionado, significa que o ZAP fica no topo de todas as janelas sempre que se chega a um ponto de interrupção.

Modo 'botões de Break'

Isto controla se haverá um só botão de break combinado na Barra de ferramentas superior ou botões separados para requests e respostas.
