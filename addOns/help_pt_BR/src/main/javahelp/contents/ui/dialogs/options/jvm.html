Tela de opções de JVM

Esta tela permite que você configure as opções da JVM usadas ao iniciar o ZAP.

Opções da JVM

Um campo de texto de formato livre que será adicionado para a chamada de linha de comando Java ao invocar ZAP através de zap.sh ou zap.bat.

A opção foi adicionada para que o tamanho de pool de alocação de memória máxima de Java possa ser definido, que é da forma: -Xmxnn Onde n é o tamanho em bytes.
Bons valores para esse campo podem ser:

  • -Xmx256m
  • -Xmx512m
  • -Xmx1024m

Diferentemente de outras opções o ZAP, estas são mantidas no arquivo ". ZAP_JVM.Properties"no diretório ZAP do usuário padrão, que depende do sistema operacional sendo usado:

  • Windows 7/8: C:\Users\<username>\OWASP ZAP
  • Windows XP: C:\Documents and Settings\<username>\OWASP ZAP
  • Linux: ~/.ZAP
  • Mac OS: ~/Library/Application Support/ZAP
Se você cometer um erro ao definir essas opções então ZAP pode falhar ao iniciar.
Se isso acontecer em seguida, excluir este arquivo deve corrigir o problema.

Observe que essas opções não são utilizadas quando se inicia o ZAP através do exe do Windows.
