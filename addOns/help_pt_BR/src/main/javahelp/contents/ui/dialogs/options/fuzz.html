Tela de opções do Fuzzer

Esta tela permite a você configurar as opções de fuzzing :

Categoria default

É a categoria a ser selecionada inicialmente e exibida quando o diálogo de Fuzz for exibido.

Varreduras simultâneas por host

Número de varreduras simultâneas a serem feitas em cada host.
Aumentar o número vai acelerar a verredura mas acrescentará sobrecarga tanto no computador onde o ZAP estiver rodando quanto no host alvo.

Inserir arquivo de fuzz customizado

Permite a você acrescentar seus próprios oarquivos para serem utilizados no fuzzing.
Eles devem ser arquivos de texto com apenas um ataque por linha.
Os arquivos são gravados no diretório 'fuzzers', abaixo do diretório home do ZAP.
