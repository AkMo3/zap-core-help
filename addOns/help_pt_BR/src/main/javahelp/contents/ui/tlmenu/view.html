Menu Visualização

Este menu manipula as opções de exibição.

Habilitar Imagens no Histórico

Se selecionado, as imagens serão exibidas na Aba de histórico.

Show Tab

Isso permite a você mostrar qualquer uma das abas disponíveis, incluindo aquelas que você ocultou.

Show All Tabs

This allows you to show all tabs.

Hide Unpinned Tabs

This allows you to hide all tabs that are not pinned.

Pin All Visible Tabs

This allows you to pin all visible tabs.

Unpin All Tabs

This allows you unpin all tabs.

Observe que Complementos podem acrescentar outros itens de menu.
