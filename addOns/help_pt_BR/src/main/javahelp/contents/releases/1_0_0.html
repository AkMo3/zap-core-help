Release 1.0.0

Este release trouxe as seguintes mudanças:

Alterações significativas:

Telas de ajuda

As telas de ajuda foram acrescentadas para todas as funcionalidades proporcionadas pelo ZAP.
Foi também implementada ajuda sensível a contexto, de modo que quando você aperta a tecla F1 aparecerá a tela de ajuda associada justamente à aba que você selecionou para trabalhar.

Pontos de interrupção

A implementação de Break Points (pontos de interrupção ou pausa) foi alterada para aproximar-se mais daquela proporcionada pelos Ambientes Integrados de Desenvolvimento de software.
Os controles de break point foram movidos para a nova Barra de Ferramentas Superior.
Os break points agora também podem ser criados clicando-se com o botão direito nas abas de Sites e Histórico.

Menu de Relatório

O Menu de Relatório foi bastante alterado e agora inclui os seguintes itens:
    Gerar Relatório...Que agora gera os relatórios solicitados
    Exportar Mensagens para o Arquivo...Movido do Menu Arquivos
    Exportar Respostas para o Arquivo...Movido do Menu Arquivos
    Exportar TODAS as URLs para o Arquivo...Nova funcionalidade
    Comparar com outra sessão...Nova funcionalidade

Busca de histórico aperfeiçoada

A nova Aba de Busca permite que você busque expressões regulares em todas as URLs, solicitações e respostas.
Os botões Seguinte e Anterior permitem a você visualizar os termos localizados nas Abas de Solicitações e Respostas.

Filtro de Histórico

A Barra de Ferramentas do Filtro de Histórico foi acrescentada para permitir que você restrinja as requests (solicitações) a serem exibidas.
As requests podem ser filtradas com base nos métodos HTTP, códigos de resultados HTTP e também em quaisquer tags, alertasou notas que tenham sido aplicadas.

Notas

Você pode associar notas com qualquer request.

Tags múltiplas

Agora você pode associar múltiplas tags com qualquer request.

Mais controle sobre os alertas

O diálogo Acrescentar Alerta permite a você acrescentar ou alterar um alerta manualmente.

Novo encoder/decoder

Um novo diálogo de Codificar/Decodificar/Hash substituiu o Encoder/decoder anterior.

Varredura passiva

A varredura passiva foi incluída.
Neste release o ZAP usará apenas varredura passiva para acrescentar automaticamente tags.
Em futuros releases ela poderá também levantar alertas para problemas em potencial.

Pequenas mudanças:

Barra de Ferramentas do Nível Superior

A Barra de Ferramentas do Nível Superior foi acrescentada, oferecendo um conjunto de botões de setup para funcionalidades de uso frequente.

Rodapé com sumário de alertas

Alertas Altos, Médios, Baixos e Informacionais agora são exibidos no rodapé.

Novo 'look & feel'

Foi utilizado um novo look & feel 'Nimbus'.

Manipulação de password de proxy

A tela de Opções de Conexão tem uma nova opção que é 'Pedir credenciais do proxy no start up'.
Se esta opção estiver selecionada, então a sua senha de proxy não será armazenada em disco.

Reenviar da aba de Sites

A Aba de Sites tem um novo item de menu no botão direito do mouse. Ele permite a você reenviar o request depois de fazer alguma modificação que deseje.

Diretório de usuário mantido entre sessões

O último diretório utilizado pelo usuário agora é armazenado no arquivo de config, de modo que fica preservado entre uma sessão e outra.
