الإصدار 1.0.0

تم إجراء التغييرات التالية في هذا الإصدار:

التغييرات الهامة:

شاشات مساعدة

مساعدة تم إضافة شاشات لجميع الوظائف التي تتيحها الآن انطلق.
كما نفذت تعليمات حساسة للسياق حيث أنه سيتم عرض عند الضغط على المفتاح F1 ثم شاشة التعليمات المقترنة بعلامة التبويب التي قمت بتحديدها.

نقاط الفصل

تنفيذ نقاط الفصل تم تغيير لتكون أقرب إلى تلك التي تقدمها برنامج بيئات التطوير المتكاملة.
كسر النقطة ضوابط تم نقل جديدة مستوى أعلى شريط الأدوات.
نص الترجمة كسر نقطة يمكن الآن أيضا أن يتم تعيين عن طريق النقر على الحق في العقد المواقع و التاريخ علامات التبويب.

تقرير القائمة

على تقرير القائمة تم تغير كبير و يشمل الآن القائمة التالية عناصر:
    إنشاء تقرير Xml...نص الترجمة الذي يولد الآن تقارير عن الطلب
    تصدير الرسائل إلى ملف.انتقلت من القائمة ملف
    تصدير الرسائل إلى ملف.انتقلت من القائمة ملف
    تصدير الرسائل إلى ملف.وظائف جديدة
    مقارنة مع آخر جلسة...وظائف جديدة

تحسن تاريخ البحث

الجديد ألسنة البحث يسمح لك للبحث عن التعبيرات العادية في جميع عناوين url الطلبات والاستجابات.
أزرار التالي والسابق تسمح لك لعرض وجدت مصطلحات البحث في طلب و رد علامات التبويب.

السجل لـ

نص الترجمة تاريخ تصفية شريط الأدوات وقد أضاف والذي يسمح لك لتقييد ما هي الطلبات التي يتم عرضها.
الطلبات يمكن تصفيتها استنادا إلى أساليب HTTP, HTTP نتيجة رموز ، و أي وسوم, تنبيهات، أو ملاحظات التي تم تطبيقها.

مُلاحظات

يمكنك الآن مشارك ملاحظات مع أي طلب.

مصابيح LED متعددة

يمكنك الآن مشارك وسوم مع أي طلب.

More control over the alerts

على Add Alert dialog allows you to manually add or change an alert.

New encoder/decoder

A new Encode/Decode/Hash has replaced the previous Encoder/Decoder.

المسح السلبي

المسح السلبي قد تم تجاوزه.
In this release ZAP will only use passive scanning for automatically adding وسوم.
In future releases it may also raise تنبيهات for potential issues.

Minor changes:

مستوى أعلى شريط الأدوات

مستوى أعلى شريط الأدوات has been added which provides a set up buttons for commonly used functionality.

Alert footer summaries

Counts of the High, Medium, Low and Informational تنبيهات are now displayed in the تذييل.

New look and feel

A new look and feel 'Nimbus' is used.

Proxy password handling

The Options Connection screen has a new option to 'Prompt for proxy credentials on start up'.
If this option is selected then your proxy password will not be stored on the hard disk.

Resend from the Sites tab

على علامة التبويب مواقع has a new right click menu item that allows you to resend the request after making any changes to it that you want to.

User directory maintained between sessions

The last directory used by the user is now stored in the config file so that it is maintained between sessions.
