Tingkat Atas Toolbar

Toolbar ini menyediakan satu set kontrol yang umum digunakan fungsi.

Mode pulldown

Hal ini memungkinkan anda untuk mengubah arus mode.

  Sesi Baru

Tombol ini sama dengan File menu 'Sesi Baru' item menu.

  buka sesi

Tombol ini sama dengan File menu 'Sidang Terbuka' item menu.

  Sesi Bertahan...

Tombol ini sama dengan File menu 'Bertahan Sesi...' item menu.

  Sesi Snapshot

Tombol ini sama dengan menu Berkas menu item 'Snapshot Session'.

  Sesi Properti...

Tombol ini sama dengan menu Berkas menu item 'Properties...'.

  Opsi...

Tombol ini sama dengan menu Perkakas menu item 'Options...'.

  Tampilkan semua tab

Tombol ini memperlihatkan semua dari tab tersembunyi.

  Sembunyikan tab yang tidak disematkan

Tombol ini menyembunyikan semua tab yang 'unpinned'. Tab dapat disematkan dan dipasangi pin melalui kecil 'pin' ikon yang ditampilkan ketika tab yang dipilih.

  Tampilkan nama tab dan ikon

Tombol ini menampilkan dari nama tab.

  Memperluas Situs Tab

Ini perubahan tampilan sehingga 'pohon' jendela yang berisi Situs tab meluas untuk panjang penuh sisi kiri.
Hal ini akan mengurangi jumlah ruang yang tersedia untuk 'informasi' jendela.

  Memperluas Informasi Tab

Ini perubahan tampilan sehingga 'informasi' jendela memanjang untuk panjang penuh dari bawah.
Hal ini akan mengurangi jumlah ruang yang tersedia untuk 'pohon' jendela.

  Tata Letak Penuh

Ini perubahan tampilan sehingga tab yang dipilih mengambil layar penuh.
Hal ini berguna ketika menggunakan ZAP pada layar kecil.

  Permintaan dan Respon tab berdampingan

Ini perubahan tampilan sehingga permintaan dan respon tab berdampingan.
Hal ini meningkatkan informasi yang dapat ditampilkan tapi tidak berarti anda tidak dapat melihat kedua permintaan dan respons pada waktu yang sama.

  Permintaan yang ditunjukkan di atas Tanggapan

Ini perubahan tampilan sehingga permintaan panel ditampilkan di atas tanggapan panel.
Hal ini mengurangi informasi yang dapat ditampilkan tapi berarti anda dapat melihat kedua permintaan dan respons pada waktu yang sama.

  Permintaan dan Respon panel berdampingan

Ini perubahan tampilan sehingga permintaan panel ditampilkan ke kiri respon panel.
Hal ini mengurangi informasi yang dapat ditampilkan tapi berarti anda dapat melihat kedua permintaan dan respons pada waktu yang sama.

 /    Set / Unset istirahat di semua permintaan dan tanggapan

Ini set dan unsets 'global' break point yang akan perangkap dan tampilan berikutnya permintaan atau respon dalam Istirahat tab.
Kemudian, anda dapat mengubah setiap bagian dari permintaan atau respon yang ingin anda dan mengirimkannya ke target aplikasi dengan menekan salah satu dari 'Langkah' atau 'Lanjutkan' tombol.
Atau anda bisa tekan tombol 'Drop' tombol untuk membuang permintaan atau respon.
Anda dapat beralih di antara satu 'gabungan' tombol istirahat dan memisahkan orang-orang untuk permintaan dan tanggapan melalui Pilihan breakpoints layar

 /    Set / Unset istirahat di semua permintaan

Ini set dan unsets 'global' break point yang akan perangkap dan menampilkan permintaan berikutnya dalam Istirahat tab.
Kemudian, anda dapat mengubah setiap bagian dari permintaan yang anda inginkan dan kirimkan ke target aplikasi dengan menekan salah satu dari 'Langkah' atau 'Lanjutkan' tombol.
Atau anda bisa tekan tombol 'Drop' tombol untuk membuang permintaan.
Anda dapat beralih di antara satu 'gabungan' tombol istirahat dan memisahkan orang-orang untuk permintaan dan tanggapan melalui Pilihan breakpoints layar

 /    Set / Unset istirahat pada semua tanggapan

Ini set dan unsets 'global' break point yang akan perangkap dan tampilan berikutnya respon dalam Istirahat tab.
Kemudian, anda dapat mengubah setiap bagian dari respon yang ingin anda dan mengirimkannya ke browser anda dengan menekan salah satu dari 'Langkah' atau 'Lanjutkan' tombol.
Atau anda bisa tekan tombol 'Drop' tombol untuk membuang permintaan.
Anda dapat beralih di antara satu 'gabungan' tombol istirahat dan memisahkan orang-orang untuk permintaan dan tanggapan melalui Pilihan breakpoints layar

  Langkah

Hal ini memungkinkan terjebak permintaan atau respon untuk melanjutkan ke aplikasi atau browser anda dengan setiap perubahan yang anda buat untuk itu.
'Global' break point akan tetap diatur sehingga berikutnya permintaan atau respon juga akan tertangkap.
Tombol ini hanya aktif ketika permintaan atau respon yang terperangkap.

  Sambung

'Global' break point akan diset sehingga selanjutnya permintaan dan tanggapan tidak akan lagi tertangkap oleh ZAP kecuali anda telah mengatur break point pada Url tertentu.
Tombol ini hanya aktif ketika permintaan atau respon yang terperangkap.

  Turun

Ini tetes terjebak permintaan atau respon sehingga tidak diteruskan ke aplikasi atau browser anda.
Tombol ini hanya aktif ketika permintaan atau respon yang terperangkap.

  Menambahkan titik patah HTTP pilihan

Ini menampilkan dialog Tambah titik patah yang mengizinkan anda untuk menentukan kriteria untuk sebuah titik patah.

  Pengelola Kebijakan Pindai...

Tampilkan dialog Pengelola Kebijakan Pindai yang mengizinkan konfigurasi dari kebijakan pindai.

  Kelola Add-ons

Ini menampilkan dialog Kelola Add-ons yang mengizinkan anda untuk menemukan, memasang dan memperbarui add-ons dari pasar online.
Mengizinkan anda juga untuk mencabut add-ons.

 /    Modus Paksa Pengguna Nyala / Mati

Ini mengaktifkan dan mematikan modus paksa pengguna.
Tombol ini hanya diaktifkan saat anda sudah mendefinisikan pengguna paksa setidaknya satu konteks, yang dilaksanakan melalui dialog Sesi Konteks.

Catatan bahwa add-ons dapat menambahkan tombol tambahan.
