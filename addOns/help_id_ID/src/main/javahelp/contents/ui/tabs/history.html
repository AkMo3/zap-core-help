Riwayat Tab

Tab Sejarah menampilkan daftar semua permintaan dalam urutan yang mereka dibuat.
Setiap permintaan Anda dapat Melihatnya:
     Indeks permintaan - setiap permintaan yang diberi nomor, dimulai dari 1
     Metode HTML, misalnya GET atau POST
     URL yang diminta
     Kode tanggapan HTTP
     Ringkasan singkat tentang apa arti dari kode tanggapan HTTP
     Lamanya seluruh waktu permintaan mengambil
     Setiap Peringatan pada permintaan tersebut
     Setiap Catatan anda telah menambahkan ke permintaan
     Setiap Tag pada permintaan tersebut
Memilih sebuah permintaan yang akan menampilkannya di Tab permintaan dan Tab tanggapan atas.

Alat penyaring bar

Alat penyaring bar yang disediakan memungkinkan anda membatasi permintaan yang ditampilkan.
Mengklik pada Tombol penyaring yang ditampilkan Sejarah dialog filter.
Ringkasan filter saat ini yang diterapkan untuk ditampilkan di sebelah kanan tombol.

Menu klik kanan

Mengklik kanan pada sebuah node akan memunculkan menu yang memungkinkan anda untuk:

Serang

Menu Attack memiliki beberapa submenu berikut:

Aktif Scan...

Ini akan meluncurkan Scan Aktif dialog yang memungkinkan anda untuk memulai sebuah aktif scan dengan titik awal sesuai dengan permintaan yang anda pilih.

Laba-laba...

Ini akan meluncurkan Laba-laba dialog yang memungkinkan anda untuk memulai laba laba dengan titik awal sesuai dengan permintaan yang anda pilih.

Sertakan dalam Konteks

Dengan menu ini anda dapat memasukkan node yang dipilih dan semua bawahannya di tempat yang ditentukan konteks.
Anda juga memiliki pilihan untuk membuat sebuah konteks yang baru.
Itu Konteks Sesi dialog akan ditampilkan untuk memungkinkan anda untuk membuat perubahan tambahan.

Kecualikan dari Konteks

Dengan menu ini anda dapat mengecualikan node yang dipilih dan semua bawahan mereka dari yang ditentukan konteks.
Itu Konteks Sesi dialog akan ditampilkan untuk memungkinkan anda untuk membuat perubahan tambahan.

Tandai sebagai konteks

Menu ini memiliki beberapa submenu untuk masing-masing konteks anda telah mendefinisikan:

Nama konteks Permintaan Login Auth Form berbasis

Ini mengidentifikasi simpul yang ditentukan sebagai permintaan masuk untuk konteks yang ditentukan.
Anda mungkin hanya memiliki satu simpul yang diidentifikasi seperti itu dalam satu konteks.
Itu Otentikasi Konteks Sesi layar akan ditampilkan untuk memungkinkan anda untuk melakukan perubahan tambahan.

Nama konteks Data digerakkan node

Ini mengidentifikasi node yang ditentukan sebagai Konten yang digerakkan data untuk konteks yang ditentukan.
Itu Konteks sesi struktur layar akan ditampilkan untuk memungkinkan anda untuk membuat setiap perubahan tambahan.

Keluarkan dari

Menu ini memiliki beberapa submenu berikut:

Proxy

Ini akan mengecualikan node yang dipilih dari proxy. Mereka masih akan di proxy melalui ZAP tapi tidak akan ditampilkan di tab mana pun.
Ini dapat digunakan untuk mengabaikan URL yang anda tahu yang tidak relevan dengan sistem yang sedang anda uji.
Simpul bisa dimasukkan lagi melalui Properti Sesi dialog

Pemindai

Ini akan mencegah agar simpul yang dipilih tidak aktif dipindai.
Simpul bisa dimasukkan lagi melalui Properti Sesi dialog

Laba-laba

Ini akan mencegah simpul yang dipilih agar tidak dijelajahi.
Simpul bisa dimasukkan lagi melalui Properti Sesi dialog

Tangani Tags...

Ini akan memunculkan Mengelola tag dialog yang memungkinkan anda untuk mengubah itu tag terkait dengan permintaan.

Catatan...

Ini akan memunculkan Tambahkan Catatan dialog yang memungkinkan anda untuk merekam catatan terkait dengan permintaan.

Hapus

Ini akan menghapus node dan semua anak-anaknya dari ZAP.
Namun mereka dapat ditambahkan kembali, untuk mencegah penggunaan menu 'Kecualikan dari' ini.

Istirahat...

Ini akan memunculkan Tambahkan titik temu dialog yang memungkinkan anda untuk menetapkan titik temu pada URL itu.

Lansiran untuk simpul ini

Jika URL yang dipilih telah peringatan terkait dengan itu kemudian mereka akan terdaftar di bawah menu ini.
Memilih salah satu peringatan akan menyebabkannya ditampilkan.

Kirim ulang...

Ini akan memunculkan Kirim ulang dialog yang memungkinkan anda untuk mengirim ulang permintaan tersebut setelah melakukan setiap perubahan yang anda inginkan.

Peringatan Baru...

Ini akan memunculkan Menambahkan dialog peringatan yang memungkinkan anda untuk merekam yang baru secara manual waspada melawan permintaan ini.

Tampilkan di tab situs

Ini akan menampilkan pesan yang dipilih di Tab situs.

Buka URL di Browser

Ini akan membuka URL simpul yang dipilih di browser default anda.

Buat formulir tes anti CSRF

Ini akan membuka URL yang akan memberi anda formulir yang dihasilkan untuk pengujian masalah CSRF.
Ini hanya akan diaktifkan untuk permintaan POST, jika API itu diaktifkan dan jika Java mendukung pembukaan URL di browser pada platform anda.
