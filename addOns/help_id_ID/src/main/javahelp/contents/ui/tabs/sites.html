Tab situs

Tab Sites menampilkan semua URL yang dikunjungi dalam struktur pohon.
anda dapat memilih salah satu nodus di pohon untuk menampilkan permintaan dan tanggapan untuk URL tersebut di tab yang relevan.

Menu klik kanan

Mengklik kanan pada sebuah Node akan memunculkan menu yang akan memungkinkan Anda untuk:

Serang

Menu Attack memiliki beberapa submenu berikut:

Scan aktif...

Ini akan meluncurkan Scan Aktif dialog yang memungkinkan Anda untuk memulai sebuah aktif scan dengan titik awal sesuai dengan permintaan yang Anda pilih.

Laba-laba...

Ini akan meluncurkan Laba-laba dialog yang memungkinkan Anda untuk memulai laba laba dengan titik awal sesuai dengan permintaan yang Anda pilih.

Sertakan dalam konteks

Dengan menu ini Anda dapat memasukkan node yang dipilih dan semua bawahannya di tempat yang ditentukan konteks.
Anda juga memiliki pilihan untuk membuat konteks baru.
Itu Konteks sesi dialog akan ditampilkan untuk memungkinkan anda membuat perubahan tambahan.

Kecualikan dari konteks

Dengan menu ini Anda dapat mengecualikan simpul yang dipilih dan semua bawahannya dari yang ditentukan konteks.
Itu Konteks sesi dialog akan ditampilkan untuk memungkinkan Anda membuat perubahan tambahan.

Tandai sebagai konteks

Menu ini memiliki beberapa submenu untuk masing - masing konteks anda telah mendefinisikan:

Nama konteks Permintaan Login Auth Form berbasis

Ini mengidentifikasi simpul yang ditentukan sebagai permintaan masuk untuk konteks yang ditentukan.
Anda mungkin hanya memiliki satu simpul yang diidentifikasi seperti itu dalam satu konteks.
Itu Otentikasi Konteks Sesi layar akan ditampilkan untuk memungkinkan Anda melakukan perubahan tambahan.

Nama konteks Data digerakkan node

Ini mengidentifikasi node yang ditentukan sebagai Konten yang digerakkan data untuk konteks yang ditentukan.
Itu Konteks sesi struktur layar akan ditampilkan untuk memungkinkan Anda melakukan perubahan tambahan.

Keluarkan dari

Menu ini memiliki beberapa submenu berikut:

Proxy

Ini akan mengecualikan node yang dipilih dari proxy. Mereka masih akan di proxy via ZAP tapi tidak akan ditampilkan di tab mana pun.
Ini bisa digunakan untuk mengabaikan URL yang anda tahu tidak relevan dengan sistem yang sedang anda uji.
Simpul bisa dimasukkan lagi melalui Properti Sesi dialog

Pemindai

Ini akan mencegah agar simpul yang dipilih tidak aktif dipindai.
Simpul bisa dimasukkan lagi melalui Properti Sesi dialog

Laba laba

Ini akan mencegah simpul yang dipilih agar tidak dijelajahi.
Simpul bisa dimasukkan lagi melalui Properti Sesi dialog

Hapus

Ini akan menghapus node dan semua anak-anaknya dari ZAP.
Namun mereka dapat ditambahkan kembali, untuk mencegah penggunaan menu 'Kecualikan dari' ini.

Istirahat...

Ini akan memunculkan jendela baru yang memungkinkan Anda mengaturnya break point pada URL itu.
Titik impas didefinisikan melalui ekspresi reguler. Jika anda mengunjungi URL yang cocok dengan ekspresi ini, ZAP akan mencegatnya dan memungkinkan anda untuk mengubah permintaan dan/atau responsnya.

Lansiran untuk simpul ini

Jika URL yang dipilih telah peringatan terkait dengan itu kemudian mereka akan terdaftar di bawah menu ini.
Memilih salah satu peringatan akan menyebabkannya ditampilkan.

Kirim ulang...

Ini akan memunculkan Kirim ulang dialog yang memungkinkan anda mengirim ulang permintaan tersebut setelah melakukan perubahan apa pun yang anda inginkan.

Peringatan Baru...

Ini akan memunculkan Menambahkan dialog lampiran yang memungkinkan Anda untuk merekam secara manual yang baru waspada melawan permintaan ini

Tampilkan di tab Riwayat

Ini akan menunjukkan simpul yang dipilih di Tab riwayat.

Tampilan dalam Browser

Ini akan membuka URL simpul yang dipilih di browser default anda.

Buat formulir tes anti CSRF

Ini akan membuka URL yang akan memberi anda formulir yang dihasilkan untuk pengujian masalah CSRF.
Ini hanya akan diaktifkan untuk permintaan POST, jika API itu diaktifkan dan jika Java mendukung pembukaan URL di browser pada platform anda.

Refresh situs pohon

Terkadang pohon Sites tidak ditampilkan dengan benar - opsi ini akan menggambar ulangnya.
