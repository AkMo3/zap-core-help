Tab break points

Break Poin tab menunjukkan anda semua break point yang telah kamu tetapkan.
Dapat diatur melalui Histori dan Situs tab serta tombol 'Tambahkan kebiasaan HTTP break point' pada toolbar level tertinggi

Menu klik kanan

Klik kanan pada sebuah titik tertentu akan memunculkan sebuah menu yang mengizinkan anda untuk:

Sunting...

Ini akan memunculkan 'Sunting break point' dialog yang memungkinkan anda untuk mengedit kriteria yang mendefinisikan break point. Mengklik ganda pada break point juga akan menampilkan dialog ini.

Hilangkan

Ini akan menghapus break point
