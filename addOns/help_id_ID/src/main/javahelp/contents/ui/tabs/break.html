Tab pecahan

Istirahat tab memungkinkan anda untuk mengubah permintaan atau tanggapan ketika telah tertangkap oleh ZAP melalui break point.
Hal ini memungkinkan anda untuk mengubah unsur-unsur yang anda biasanya tidak akan dapat mengubah melalui browser anda, termasuk:
     Header
     Bidang tersembunyi
     Bidang cacat
     Bidang-bidang yang menggunakan javascript untuk menyaring karakter ilegal
Fungsi ini adalah kunci untuk secara efektif pena pengujian aplikasi anda.

2 panel hanya akan berisi apa saja jika ZAP telah menangkap permintaan atau tanggapan.
Anda dapat mengubah apa pun dalam 2 panel dan kemudian meneruskan permintaan atau tanggapan yang menggunakan tombol-tombol pada Tingkat Atas Toolbar.

Pull downs memungkinkan anda untuk memilih yang berbeda Pandangan untuk permintaan atau tanggapan header dan body.

Tekan pilihan kanan

Mengklik kanan node akan memunculkan menu yang akan memungkinkan anda untuk:

Cari...

Ini akan memunculkan Temukan percakapan.

Encode/Decode/Hash...

Ini akan memunculkan Encode/Decode/Hash.
Jika anda mempunyai beberapa teks yang menyorot maka ini akan menjadi secara otomatis termasuk dalam dialog.

Salin

Ini akan menyalin senar yang dipilih ke clipboard.
