Tab Sesi HTTP

Tab ini menunjukkan kumpulan sesi HTTP yang teridentifikasi untuk masing-masing situs, seperti yang terdeteksi oleh HTTP sesi ekstensi.

Situs saat ini informasi yang dimaksud dapat dipilih melalui alat bar atau situs tab.

Alat bar menyediakan sebuah tombol ("Sesi Baru") yang memungkinkan anda untuk memulai sesi baru, memaksa semua pesan permintaan keluar tanpa sesi token yang ditetapkan, sehingga server menganggap itu adalah sesi yang baru. Hal ini memungkinkan pembuatan sesi baru, tanpa merusak yang lama.

Setiap entri pada tabel Sesi (setiap sesi) awalnya memiliki nama yang dihasilkan, namun bisa diubah dengan memilih beri 'Nama' dan mengeditnya.

Setiap entri dalam tabel Sesi dapat diklik dengan benar, yang mengaktifkan Menu Popup, dengan pilihan berikut:

  • Menghapus sesi - sesi yang dihapus
  • Tetapkan sebagai aktif (hanya tersedia pada sesi tidak-aktif) tanda sesi ini aktif. Jika sesi sebelumnya ditetapkan sebagai aktif, itu akan menjadi tidak diatur sebagai aktif dan, jika tidak menentukan setiap nilai token, itu akan dihapus.
  • Tidak diatur sebagai aktif (hanya tersedia pada sesi aktif) - menandai sesi ini sebagai tidak aktif lagi. Jika sesi ini tidak menentukan setiap nilai token, itu akan dihapus.
Mengenai sesi aktif, rincian lebih lanjut dapat dibaca di konsep umum yang membantu halaman Sesi HTTP ekstensi

Untuk setiap sesi anda dapat melihat:
  • Aktif - Apakah ini sesi aktif atau tidak
  • Nama - Nama sesi
  • Sesi nilai Token - daftar nilai yang terkait masing-masing sesi token. Entrinya dipisahkan oleh simbol ';'.
  • Pesan Sesuai - jumlah pesan HTTP yang telah ada diimbangi perpanjangan dengan sesi ini
