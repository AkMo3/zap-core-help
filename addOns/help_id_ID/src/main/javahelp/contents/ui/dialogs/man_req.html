Manual meminta Editor dialog

Dialog ini memungkinkan kamu membuat permintaan dari awal yang akan diserahkan ke target yang ditentukan.

Permintaan tab

Ini menunjukkan permintaan header dan data, baik dalam satu atau dua panel tergantung pada opsi yang dipilih.

'Metode' tarik ke bawah memungkinkan Anda untuk beralih antara metode HTTP.
Perhatikan bahwa ketika metode berubah untuk posting maka parameter URL pindah ke dalam tubuh, dan ketika metode berubah dari posting kemudian parameter dalam tubuh pindah ke URL.

Pull down memungkinkan Anda untuk memilih berbeda Pemandangan untuk header permintaan dan tubuh.

  Split layar untuk header dan tubuh

Hal ini akan mengubah tampilan sehingga panel terpisah digunakan untuk header dan tubuh.

  Tampilan gabungan untuk header dan tubuh

Hal ini akan mengubah tampilan sehingga header dan tubuh ditampilkan dalam satu pane.

  Gunakan sesi pelacakan saat ini

Lihat 'Aktifkan sesi pelacakan (Cookie)' item menu di Sunting menu.

  Ikuti pengalihan

Jika dipilih secara otomatis mengikuti pengalihan yang dikirim ke browser.

  Permintaan dan respon tab berdampingan

Ini mengubah tampilan sehingga tab permintaan dan respons berdampingan.
Hal ini meningkatkan informasi yang dapat ditampilkan namun berarti kamu tidak dapat melihat permintaan dan tanggapan pada saat bersamaan.

  Permintaan ditampilkan diatas Tanggapan

Ini akan merubah tampilan sehingga panel permintaan ditampilkan diatas panel tanggapan.
Hal ini menurunkan informasi yang bisa ditampilkan namun berarti kamu dapat melihat permintaan dan tanggapan pada saat bersamaan.

  Panel permintaan dan tanggapan berdampingan

Ini merubah tampilan sehingga panel permintaan ditampilkan disebelah kiri panel tanggapan.
Hal ini menurunkan informasi yang dapat ditampilkan tetapi menunjukkan bahwa anda bisa melihat permintaan dan tanggapan pada waktu yang bersamaan.

Tab Tanggapan

Ini menunjukkan data yang dikirim ke browser kamu sebagai tanggapan atas permintaan yang kamu kirimkan, baik dalam satu atau dua panel tergantung pada pilihan yang dipilih.

Pull downs memungkinkan anda untuk memilih sesuatu yang berbeda Tampilan untuk Respon header dan body.

Klik kanan menu

Mengklik kanan akan memunculkan menu yang akan mengizinkan anda untuk:

Mencari...

Ini akan memunculkan Mencari dialog.

Encode/Decode...

Ini akan memunculkan Encode/Decode dialog.
Jika anda menyoroti setiap teks maka ini akan secara otomatis disertakan dalam dialog.

Diakses melalui

     Alat menu tingkat atas'Editor Permintaan Manual ...' item menu

     Dialoguntuk rincian dari dialog atau popup
