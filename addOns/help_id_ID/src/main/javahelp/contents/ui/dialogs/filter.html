Kirim ZAP session request ID

Penyaring ini akan menambahkan tag header khusus ke setiap permintaan yang dikirim ke server. Jadi, anda dapat melacak permintaan ZAP saat menyelidiki aplikasi web anda sendiri. Ini berguna untuk memiliki id yang berbeda terhadap beberapa permintaan, saat melakukan penguraian misalnya file log server HTTP.

Permintaan header telah dimodifikasi seperti ini (contoh):

X-ZAP-permintaan ID: <sessionName>– <number>
  • sessionName yaitu URL dikodekan dengan nama sesi
  • nomor diawali dengan satu; meningkat pada setiap permintaan; berlaku hanya setiap sesi
