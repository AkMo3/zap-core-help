Pilihan Global mengecualikan layar URL

Layar ini memungkinkan anda untuk mengkonfigurasi Global Exclude URL regexs pada ZAP:

Regex

Regex dari URL yang ZAP akan mengabaikan. Klik tombol Add, modifikasi, atau menghapus untuk mengedit setiap regex.

Aktifkan

Jika kotak ini dicentang, maka URL regex ini akan diabaikan oleh ZAP.
