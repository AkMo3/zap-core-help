Pilihan layar Koneksi

Sebuah Pilihan layar Koneksi mengizinkan kamu untuk mengkonfigurasi koneksi pilihan ZAP:

Waktu habis dalam hitungan detik

Ini membuat lebih mudah untuk menguji kelambatan aplikasi.

Kegagalan Agen Pengguna

Agen Pengguna harus menggunakan ZAP ketika membuat pesan HTPP (sebagai contok, penjebakan pesan atau permintaan KONEKSI untuk keluar proxy).

Pemintaan Header Cookie Single

Mengontrol jikalau cookie dikelola ZAP (Aktifkan Session Tracking (Cookie)) harus ditetapkan pada satu permintaan header "Cookie" atau beberapa permintaan header "Cookie", saat mengirim permintaan HTTP ke server.

DNS

TTL Permintaan yang Berhasil

Mendefinisikan berapa lama permintaan DNS yang berhasil harus disembunyikan:
  • Negatif angka, sembunyikan selamanya;
  • Nol, menonaktifkan persembunyian;
  • Positif angka, jumlah detik permintaan akan disembunyikan.
Catatan: Perubahan diterapkan setelah restart.

Sebuah pilihan bisa juga ditetapkan menggunakan sebuah -config argumen batas perintah dengan tombol connection.dnsTtlSuccessfulQueries.

Protokol Keamanan

Mengizinkan untuk memilih versi SSL/TLS yang diaktifkan untuk koneksi keluar (sebagai contoh, ke server). Paling sedikit satu versi harus diaktifkan, versi yang tidak didukung oleh JRE tidak akan dipilih dan dinonaktifkan.
Sebuah pilihan SSLv2Hello harus dipilih dalam hubungannya setidaknya dengan satu versi SSL/TLS.

Menggunakan rantai Proxy

Bagian ini memungkinkan Anda terhubung ke proxy lain untuk koneksi keluar.
Hal ini sering dibutuhkan di lingkungan perusahaan.

Proxy membuktikan

Bagian ini menizinkan kamu untuk mengkonfigurasi pengeluaran pembuktian proxy.
