API

ZAP menyediakan sebuah Application Programming Interface (API) yang mengizinkan anda untuk berinteraksi dengan ZAP pemrograman.

API yang tersedia di JSON, HTML dan XML format.
Sederhana UI web yang memungkinkan anda untuk menjelajahi dan menggunakan API tersedia melalui URL http://zap/ ketika anda proksi melalui ZAP, atau melalui host dan port ZAP adalah mendengarkan pada, misalnya http://localhost:8080/.

Secara default hanya mesin ZAP yang sedang berjalan yang dapat mengakses API. Anda dapat mengizinkan mesin lain, yang dapat menggunakan ZAP sebagai proxy, akses ke API. Konfigurasi API menggunakan Options API screen.

API menyediakan akses untuk sebagian besar fitur inti ZAP seperti active scanner dan spider.
Versi selanjutnya dari ZAP akan meningkatkan fungsionalitas yang tersedia melalui API.
