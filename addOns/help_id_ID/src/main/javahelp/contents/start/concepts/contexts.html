Konteks

Konteks adalah cara yang berkaitan satu set Url bersama-sama.
Anda dapat menentukan setiap konteks yang anda suka, tapi diharapkan akan sesuai dengan konteks aplikasi web.
Dianjurkan bahwa anda menentukan konteks yang baru untuk setiap aplikasi web yang membuat sistem anda menguji, dan mengatur mereka dalam cakupan seperti anda menguji masing-masing.

Konteks didefinisikan sebagai serangkaian ekspresi reguler (regex) yang diterapkan ke semua URL di Tab situs.
Anda dapat mengkonfigurasi konteks melalui:

  • Klik kanan menu item pada Situs atau tab Riwayat
  • The Session Contexts dialogs.
