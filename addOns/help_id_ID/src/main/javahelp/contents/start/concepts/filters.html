Filter

CATATAN: Filter pada dasarnya telah diganti dengan add-on Replacer dan skrip yang lebih kuat dan fleksibel.
Filter akan dihapus dalam versi ZAP.

Filter menambahkan fitur ekstra yang dapat di aplikasikan untuk setiap permintaan dan respon.
Secara bawaan tidak ada filter yang diaktifkan sejak awal. Mengaktifkan seluruh filter dapat memperlambat proxy.

Filter dikonfigurasikan melalui Filters dialog.
