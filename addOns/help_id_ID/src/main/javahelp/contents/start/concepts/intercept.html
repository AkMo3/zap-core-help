Man-in-the-middle Proxy

ZAP is a Man-in-the-middle Proxy. Ini memungkinkan anda untuk melihat seluruh permintaan yang anda buat pada aplikasi web dan seluruh respon yang anda terima darinya.
Antara lain hal ini memungkinkan anda untuk melihat panggilan AJAX yang mungkin tidak jelas.

Anda juga dapat mengatur break points yang mana memungkinkan anda untuk mengubah permintaan dan respon dengan cepat.
