URL secara global dikecualikan

Catatan: fitur ini adalah beta.

Secara global Url Dikecualikan satu set Ekspresi Reguler (regex) bahwa ZAP benar-benar mengabaikan seluruh aplikasi. URL ini tidak akan muncul di Proxy, Scanner, atau Spider untuk ZAP. Selain itu, regex URL akan disimpan dalam konfigurasi global untuk ZAP dan bukan di dalam file sesi. Ini Url yang akan bertahan antara setiap penggunaan ZAP.

Regex untuk URL dikonfigurasikan menggunakan Global Exclude URL screen pada pilihan ZAP.
