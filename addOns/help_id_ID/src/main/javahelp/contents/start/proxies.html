Penguasaan konfigurasi

Anda perlu untuk mengkonfigurasi browser Anda untuk menggunakan ZAP sebagai persetujuan.
Standarnya ZAP menggunakan alamat dari 'host lokal ' dan cara kerja dari '8080', akan tetapi ini dapat diubah melalui persetujuan lokal pilihan layar. Teks untuk Terjemahan Peraturan untuk versi terbaru browser yang paling umum digunakan:

Warna kuning pada jendela

    Tekanlah menyesuaikan tombol kanan atas dan mengontrol google chrome
    Pilih lah pilihan menu pada item
    Tekanlah ganti tombol petunjuk
    Tekan lah pengaturan tombol LAN
    Ikutilah berikut ini adalah pengaturan LAN

Pada jendela berwarna kuning

    Pilih lah peralatan menu
    Agar memilih pilihan menu item
    Tekan lah memajukan tombol
    Pilih lah tab jaringan
    Menekan tombol pengaturan
    Pilih lah petunjuk manual untuk konfigurasi tombol radio
    Masukan di 'HTTP Proxy:' field 'Alamat' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Masukkan dalam 'Port' kanan 'HTTP Proxy' lapangan 'Port' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Tekan Pengaturan koneksi tombol 'OK'
    Tekan Pilihan tombol 'OK'

Firefox (di Linux)

    Pilih'Edit' menu
    PilihPilih'Preferensi' item menu
    Ikutipetunjuk di atas dari tombol 'Advanced'

Firefox (pada OS X)

    Pilih Menu 'Firefox'
    Pilih 'Preferensi...' item menu
    Tekan tombol 'Advanced'
    Ikuti petunjuk di atas dari tombol 'Advanced'

Internet Explorer 7 dan 8

    Pilih Menu 'Tools'
    Pilih 'Internet Options' item menu
    Pilih Tab 'Koneksi'
    Tekan Tombol 'Pengaturan LAN'
    Ikuti 'Windows LAN Settings' bagian bawah

Opera

    Pilih 'Menu' di sudut atas sebelah kiri
    Pilih Item menu 'Pengaturan'
    Pilih 'Preferensi' item menu
    Pilih Memajukan Tombol
    Pilih 'Jaringan' item di tangan kiri daftar
    Tekan tombol 'Proxy Servers'
    Pilih kotak centang 'HTTP'
    Masukkan di bidang ke kanan dari 'HTTP', 'Alamat' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Masukkan di HTTP 'Port' lapangan 'Port' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Tekan Proxy server tombol 'OK'
    Tekan Preferensi tombol 'OK'

Safari (OS X)

    Tekan 'Setelan Safari' tombol di sudut kanan atas
    Pilih 'Preferensi' item menu
    Pilih 'Advanced' tab
    Tekan 'Mengubah Pengaturan' tombol di sebelah 'Proxy' label
    Ikuti 'OS X Sistem pengaturan Proxy' bagian bawah

Safari (pada Windows)

    Tekan 'Menampilkan menu general setelan Safari' tombol di sudut kanan atas
    Pilih 'Preferensi' item menu
    Pilih 'Advanced' tab
    Tekan 'Mengubah Pengaturan' tombol di sebelah 'Proxy' label
    Tekan 'LAN Settings' tombol
    Ikuti 'Windows LAN Settings' bagian bawah

OS X Sistem pengaturan Proxy

    Pilih 'Web Proxy (HTTP)
    Masukkan di 'Alamat:' field 'Alamat' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Masukkan di 'Port' lapangan 'Port' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Tekan Proxy Setting tombol 'OK'
    Tekan Jaringan Pilihan 'Terapkan' tombol

Jendela Pengaturan LAN

    Pilih 'Gunakan server proxy untuk LAN anda' checkbox
    Masukkan di 'Alamat:' field 'Alamat' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Masukkan di 'Port' lapangan 'Port' anda dikonfigurasi dalam Pilihan Proxy Lokal layar
    Tekan Local Area Network (LAN) Pengaturan tombol 'OK'
    Tekan Internet Pilihan tombol 'OK'
