Memulai

Cara tercepat untuk mendapatkan pergi dengan ZAP adalah dengan menggunakan Quick Start add-on, yang diinstal secara default.
Hal ini memungkinkan anda untuk memasukkan URL yang ZAP akan pertama laba-laba dan kemudian aktif scan.
Untuk lebih mendalam dalam tes yang harus anda memulai aplikasi anda menggunakan browser anda atau otomatis uji regresi sedangkan proksi melalui ZAP.

Pada inti ZAP adalah man-in-the-middle proxy.
Untuk mendapatkan sesuatu yang maksimal dari ZAP anda perlu mengkonfigurasikan peramban atau fungsional tes untuk terhubung ke aplikasi web yang ingin anda uji melalui ZAP.
Jika dibutuhkan, anda juga bisa mengkonfigurasi ZAP untuk terhubung melalui proxy lain - hal ini sering diperlukan di lingkungan perusahaan.

Anda dapat dengan cepat dan mudah meluncurkan browser yang pra-dikonfigurasi untuk proxy melalui ZAP melalui Cepat Mulai tab. Ini dapat meluncurkan salah satu peramban yang paling umum yang telah anda pasang dengan profil baru.
Jika anda ingin menggunakan salah satu peramban dengan profil yang telah ada, sebagai contoh dengan add-ons peramban lain yang terpasang, kemudian anda perlu mengkonfigurasi peramban secara manual ke proxy melalui ZAP.
Jika anda mengetahui bagaimana mengatur proxy pada peramban web anda, silahkan lanjutkan dan mencobanya!
Jika anda tidak yakin maka lihatlah pada bagian Configure proxies.

Setelah anda telah mengkonfigurasi ZAP sebagai proxy pramban anda kemudian coba untuk menghubungkan pada aplikasi web yang akan di tes.
Jika anda tidak dapat terhubung kemudian cek kembali pengaturan proxy anda. Anda perlu memeriksa pengaturan proxy peramban anda, dan juga pengaturan proxy ZAP.
Yang juga perlu memeriksa bahwa aplikasi yang anda mencoba untuk menguji berjalan!

Ketika anda berhasil terhubung ke aplikasi anda melalui peramban anda kemudian liat ZAP kembali. Sekarang anda akan melihat satu atau lebih baris dalam Situs dan Sejarah tab.
Jika demikian kita berada dalam bisnis. Jika tidak maka anda perlu untuk mengecek pengaturan proxy anda kembali.

Hal selanjutnya yang dilakukan untuk memulai basic penetration test.
